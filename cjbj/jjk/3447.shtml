  • 09年自考“企业管理概论”练习题（2）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、企业经营战略的正确制定和实施，要确保企业资产的保值和增值。这在企业制定经营战略要考虑的重大战略关系中，是指（ ）

　　A 企业与投资者的关系

　　B 企业与顾客的关系

　　C 企业与竞争者的关系

　　D 企业与政府的关系

　　参考答案：A

　　2、制定经营战略应重视对企业资源保证程度的研究。这在企业制定经营战略要考虑的重大战略关系中，是指（ ）

　　A 企业与投资者的关系

　　B 企业与顾客的关系

　　C 企业与竞争者的关系

　　D 企业与供应者的关系

　　参考答案：　D

　　3、企业为贯彻战略思想和实现战略目标、战略重点，所确定的企业生产经营活动应遵循的基本原则、指导规范和行动方略。这在经营战略的内容中是指（ ）

　　A 战略目标

　　B 战略思想

　　C 战略方针

　　D 战略对策

　　参考答案　C

　　4、为实现战略目标而采取的重要措施和重要手段。这在经营战略的内容中是指（ ）

　　A 战略目标

　　B 战略思想

　　C 战略方针

　　D 战略对策

　　参考答案：　D

　　5、在对企业内外环境进行深入调查研究的基础上，对市场需求、竞争状况、资源供应、企业实力、国家政策、社会要求等主要因素进行综合分析后，所确定的统率和指导企业全局和长远发展的谋划和方略。这是指（ ）

　　A 职能战略

　　B 产品战略

　　C 市场战略

　　D 总体战略

　　参考答案：　D

　　6、在企业战略中，市场战略属于（ ）

　　A 职能战略

　　B 产品战略

　　C 技术战略

　　D 总体战略

　　参考答案：A

　　7、经营战略管理是指（ ）

　　A 经营战略的制定过程

　　B 经营战略的实施过程

　　C 经营战略的控制过程

　　D 经营战略的制定、实施和控制过程

　　参考答案：　D

　　8、对战略的制定过程和实施过程自始至终进行管理。这反映了经营战略管理的 （ ）

　　A 未来性特点

　　B 全程性特点

　　C 不确定性特点

　　D 高层性特点

　　参考答案：　B

　　9、制定和实施经营战略的前提是 （ ）

　　A 企业内外环境调查

　　B 确定企业经营宗旨

　　C 拟定、评价和选择战略方案

　　D 战略方案的落实

　　参考答案：A

　　10、经营方案制定过程中的的决策阶段是指（ ）

　　A 企业内外环境调查

　　B 确定企业经营宗旨

　　C 拟定、评价和选择战略方案

　　D 战略方案的落实

　　参考答案：　C

　　11、将企业经营战略目标分解为企业各部门和下属各单位的具体目标，以便落实责任和检查监督。这在战略实施中属于（ ）

　　A 制定战略实施计划工作

　　B 确立企业战略组织工作

　　C 保证战略资源的有效配置工作

　　D 战略控制工作

　　参考答案：A

　　12、从统一领导，加强组织协调的原则出发，战略的实施应注意抓好（ ）

　　A 制定战略实施计划工作

　　B 确立企业战略组织工作

　　C 保证战略资源的有效配置工作

　　D 战略控制工作

　　参考答案：　B

　　13、在战略计划实施过程中，为了保证按战略计划的要求进行生产经营活动，所采用的不断评审实际工作、将反馈回来的实际成效信息与计划目标比较，及时发现偏差、采取纠正措施的活动。这是指 （ ）

　　A 战略计划工作

　　B 战略组织工作

　　C 战略实施工作

　　D 战略控制工作

　　参考答案：　D

　　14、开展战略控制工作的首要环节是（ ）

　　A 确定控制标准

　　B 衡量工作成效

　　C 分析原因

　　D 纠正偏差

　　参考答案：A

　　15、把权力集中于少数高层管理人员手里，以减少分层控制所造成的矛盾。这种控制属于 （ ）

　　A 直接控制

　　B 避免控制

　　C 过程控制

　　D 反馈控制

　　参考答案：　B

　　16、在战略实施中，对战略行动的结果趋势进行预测，并将预测值与既定的标准进行比较和评价，发现可能出现的偏差，从而提前采取纠偏措施。这种控制属于（ ）

　　A 直接控制

　　B 反馈控制

　　C 过程控制

　　D 前馈控制

　　参考答案：　D

　　17、对战略执行过程中产生的实际结果与预定目标有明显差距时采取的对战略方案的修改。这是指 ）

　　A 战略计划工作

　　B 战略组织工作

　　C 战略调整工作

　　D 战略控制工作

　　参考答案：　C

　　18、对涉及企业全局的基本战略方向的修改，这属于（ ）

　　A 局部性战略调整

　　B 职能战略调整

　　C 总体战略调整

　　D 组织战略调整

　　参考答案：　C

　　19、企业战略家应具有坚强的意志、稳定的情感、良好的性格、鲜明的个性。这在企业战略家的基本素质中属于 （ ）

　　A 品德高尚，志向高远

　　B 思维敏捷，知识渊博

　　C 心理健康、个性优异

　　D 足智多谋，能力超群

　　参考答案：　C

　　20、企业战略家应富有韬略，在决策、组织、协调等管理能力方面有超群的突出表现。这在企业战略家的基本素质中属于 （ ）

　　A 品德高尚，志向高远

　　B 思维敏捷，知识渊博

　　C 心理健康、个性优异

　　D 足智多谋，能力超群

　　参考答案：　D

　　21、组建企业战略领导班子，首先要选好第一把手。这在组建企业战略领导班子需要遵循的原则中属于 （ ）

　　A 选择首要领导的原则

　　B 首要领导组阁的原则

　　C 能力匹配组合的原则

　　D 合作、和谐的原则

　　参考答案：A

　　22、由已经选定的首要领导者来选择和确定战略领导班子中的其他成员。这在组建企业战略领导班子需要遵循的原则中属于

　　A 选择首要领导的原则

　　B 首要领导组阁的原则

　　C 能力匹配组合的原则

　　D 合作、和谐的原则

　　参考答案：　B

　　23、凡处于不确定性低的稳定的经营环境，实施产品单一化战略的中小企业，可选择的组织结构是

　　A 职能制组织结构

　　B 事业部制组织结构

　　C 模拟分权制组织结构

　　D 矩阵制组织结构

　　参考答案：A

　　24、变化比较大而不确定性为中等或很高的环境，实施产品多样化战略的大型企业，可选择的组织结构是 （ ）

　　A 职能制组织结构

　　B 事业部制组织结构

　　C 模拟分权制组织结构

　　D 矩阵制组织结构

　　参考答案：　B

　　（答案仅供参考）
