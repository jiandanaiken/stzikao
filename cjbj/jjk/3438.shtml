  • 09年自考“企业管理概论”综合题（8）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、企业领导体制，本质上是确定（ ）的相互关系的制度。

　　A 所有者、管理者和劳动者

　　B.所有者和管理者

　　C 所有者和劳动者

　　D.管理者和劳动者

　　答案　A

　　2、关于企业领导层的构成、职责分工、权力划分及工作机构设置等的规定属（ ）。

　　A 经济体制

　　B 政治体制

　　C 领导体制

　　D 人事制度

　　答案　C

　　3、企业领导体制的核心问题是规定企业中（ ）地位、责任 和权力。

　　A 行政组织

　　B 行政组织、党组织和职代会三者的

　　C 党组织

　　D 职代会

　　答案　B

　　4、企业的政治核心是（ ）。

　　A 职代会

　　B 行政组织

　　C 党组织

　　D 厂长（经理）

　　答案　C

　　5、企业党委的重要职责是（ ）。

　　A 调动职工的积极性

　　B 定期做企业的工作报告

　　C 制定企业经营规划

　　D 监督保证

　　答案　D

　　6、社会主义企业管理的本质特征是（ ）。

　　A 党组织领导

　　B 实行职工民主管理

　　C 厂长负责制

　　D 党、政一齐抓

　　答案　B

　　7、厂长与职代会的关系（ ）。

　　A 平等关系

　　B 隶属关系

　　C 职工无条件服从厂长的领导

　　D 厂长领导、职工当家作主

　　答案　D

　　8、我国非公司制国有企业的法定代表人是（ ）。

　　A 厂长（经理）

　　B 党委书记

　　C 全体职工

　　D 董事长

　　答案　A

　　9、发展社会主义市场经济，客观上要求企业（ ）。

　　A 实行党委一元化领导

　　B 实行厂长（经理）负责制

　　C 实行厂长（经理）一言堂

　　D 不能实行厂长（经理）负责制

　　答案　B

　　10、厂长（经理）具有（ ）。

　　A 决策权

　　B 指挥权

　　C 用人权

　　D 以上三点

　　答案　D

　　11、公司制企业是（ ）。

　　A 由职代会来经营管理的

　　B 由党代会来经营管理的

　　C 由法人治理结构来经营管理的

　　D 由民主治理结构来经营管理的

　　答案　C

　　12、法人治理结构由（ ）。

　　A 以下三点共同构成

　　B 股东会和董事会

　　C 以经理为首的执行机构

　　D 监事会

　　答案　A

　　13、企业领导体制是（ ）。

　　A 以民主集中制为特征的

　　B 以它为统一领导为特征的

　　C 以经理为首的执行机构为特征的

　　D 以法人治理结构为特征的

　　答案　D

　　14、公司的最高权力机构是（ ）。

　　A 经理

　　B 董事

　　C 股东大会

　　D 党委

　　答案　C

　　15、利润分配权属（ ）。

　　A 董事会

　　B 股东大会

　　C 经理

　　D 党委

　　答案　B

　　16、董事会由（ ）。

　　A 股东大会选举产生

　　B 党委讨论通过

　　C 经理确定

　　D 董事长提名

　　答案　A

　　17、董事会（ ）。

　　A 代表职工利益，执行职代会的决

　　B 代表职工利益，执行厂长的决策

　　C 代表股东利益，执行股东会的决议

　　D 代表股东利益，执行董事长的决策

　　答案　C

　　18、行政管理系统由（ ）。

　　A 经理、副经理组成

　　B 经理、副经理、总会计师组成

　　C 经理、副经理、总会计师、总工程师组成

　　D 经理、副经理、总会计师、总工程师、职代会代表组成

　　答案　C

　　19、公司经理由（ ）。

　　A 董事会聘任

　　B 党委任命

　　C 职代会选举

　　D 上级委派

　　答案　A

　　20、监事会是（ ）。

　　A 执行机构

　　B 监督机构

　　C 决策机构

　　D 人事机构

　　答案　B

　　21、监事会（ ）。

　　A 隶属于董事会

　　B 是董事会的上层机构

　　C 与股东大会并立

　　D 与董事会并立

　　答案　D

　　22、监事会（ ）。

　　A 对董事无监督权力

　　B 对经理无监督权力

　　C 对董事、经理有监督权力

　　D 对董事有监督权力、对经理无监督权力

　　答案　C

　　（答案仅供参考）
