  • 09年自考“企业管理概论”综合题（5）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、现代企业管理应（ ）。

　　A 以人为本

　　B 以物质为本

　　C 以利润为本

　　D 以企业为本

　　答案　A

　　2、企业活力的源泉，（ ）。

　　A 来自于物质奖励

　　B 来自于领导层的决策和指挥

　　C 来自于人

　　D 来自于竞争

　　答案　C

　　3、企业是一个独立的经济实体，同时（ ）。

　　A 独立于社会之外

　　B 是一个政治组织

　　C 是社会系统的组成部分

　　D.是一个独立的社会系统

　　答案　C

　　4、企业的使命是创造经济效益，（ ）。

　　A 同时也创造社会效益

　　B 而不创造社会效益

　　C 但与社会无关系

　　D 和赞助社会一切公共事业

　　答案　A

　　5、在企业管理体系中，最重要的管理层是（ ）。

　　A 职能层

　　B 基层

　　C 中层

　　D 战略层

　　答案　D

　　6、企业战略层管理的核心内容（ ）。

　　A 制定和组织实施财务工作

　　B 制定和组织实施人事工作

　　C 制定和组织实施经营战略

　　D 制定和组织实施销售工作

　　答案　C

　　7、企业中层管理，（ ）。

　　A 是高层管理和基层管理连接的纽带

　　B 是企业的决策者

　　C 是企业的最直接的运作者

　　D 是企业的行政管理

　　答案　A

　　8、企业中层管理，（ ）。

　　A 是高层管理的参谋和助手

　　B 对基层管理进行指导和监督

　　C 是专业性管理

　　D 以上三点

　　答案　D

　　9、企业基层管理，（ ）。

　　A 对象是职工

　　B 对象是作业层

　　C 对象是产品

　　D 对象是服务

　　答案　B

　　10、现场作业管理，（ ）

　　A 包括工序、物流、环境及基层组织管理

　　B 完成基层承担的作业任务

　　C 其他三点

　　D 实现高层确定的经营目标

　　答案　C

　　11、工业企业的基本活动（ ）。

　　A 是生产活动

　　B 是销售活动

　　C 是经营活动

　　D 是决策活动

　　答案　A

　　12、生产技术管理（ ）。

　　A 是搞好决策和指挥的技术保证与强大后盾

　　B 是搞好销售和服务的技术保证与强大后盾

　　C 是搞好经营和预测的技术保证与强大后盾

　　D 是搞好生产和市场营销的技术保证与强大后盾

　　答案　D

　　13、采购、储备、保管、发放和合理使用（ ）。

　　A 经营管理

　　B 物资供应管理

　　C 销售管理

　　D 市场管理

　　答案　B

　　14、物资供应管理（ ）。

　　A 是保证生产经营正常运转的重要工作

　　B 是保证销售正常运转的重要工作

　　C 是保证市场正常运转的重要工作

　　D 是保证组织正常运转的重要工作

　　答案　A

　　15、企业的市场营销（ ）。

　　A 是行政管理活动

　　B 是物资管理活动

　　C 是销售管理活动

　　D 是商务活动

　　答案　D

　　16、企业的市场营销包括（ ）。

　　A 市场研究、广告宣传

　　B 订货合同、销售渠道

　　C 其他三点

　　D 为用户服务

　　答案　C

　　17、资金的运动过程（ ）。

　　A 伴随着控制和决策过程

　　B 伴随着生产经营过程

　　C 伴随着人事管理过程

　　D 伴随着物资运转过程

　　答案　B

　　18、财务管理（ ）。

　　A 包括资金筹措

　　B 包括固定和流动资金管理

　　C 包括成本费用和利润管理

　　D 以上三点

　　答案　D

　　19、人力资源管理（ ）。

　　A 属于行政管理

　　B 属于专业管理

　　C 属于机关管理

　　D 属于企业管理

　　答案　B

　　20、人力资源管理（ ）。

　　A 以下三点

　　B 包括对人员的招收、录用

　　C 包括对人员的调配、考核、培训

　　D 包括对人员的升迁

　　答案　A

　　（答案仅供参考）
