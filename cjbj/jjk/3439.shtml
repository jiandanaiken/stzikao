  • 09年自考“企业管理概论”综合题（9）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、企业组织机构的设置，必须以（ ）。

　　A 企业的决策为依据

　　B 企业的战略任务和经营目标为依据

　　C 企业的人员构成为依据

　　D 企业的领导意愿为依据

　　答案　B

　　2、组织机构和任务目标的关系为（ ）

　　A 组织机构是目的，任务目标是手段

　　B 组织机构是目的，任务目标也是目的

　　C 组织机构是手段，任务目标是目的

　　D 组织机构是手段，任务目标也是手段

　　答案　C

　　3、组织机构设置是否合理，其最终标准是（ ）。

　　A 能否促进企业任务和目标的实现

　　B 能否按照领导意图有效工作

　　C 能否调动职工积极性

　　D 能否合理安排工作内容

　　答案　A

　　4、企业管理应实行（ ）。

　　A 注重专业分工原则

　　B 注重专业分工与协作原则

　　C 注重协作原则

　　D 注重统一原则

　　答案　B

　　5、企业管理实行分工，依据（ ）

　　A 领导意图

　　B 组织安排

　　C 人员特长

　　D 专业特点

　　答案　D

　　6、搞好分工应（ ）。

　　A 越细越好

　　B 分工适当

　　C 简单合并科室

　　D 简单拆分科室

　　答案　B

　　7、加强协作，应（ ）。

　　A 实行系统管理，创造协调的环境

　　B 设立临时或常设的委员会、会议等

　　C 采取矩阵组织结构，制定工作流程

　　D 以上三点

　　答案　D

　　8、机构设置应（ ）。

　　A 实行首长负责制

　　B 实行多头领导

　　C 实行行政命令与生产经营脱钩

　　D 实行大家说了算

　　答案　A

　　9、实行直线――参谋制，要求（ ）。

　　A 有直线指挥人员

　　B 参谋作为指挥人员

　　C 有直线指挥人员和参谋职能人员

　　D 参谋和助手对下级发布命令和指挥

　　答案　C

　　10、管理幅度是指（ ）。

　　A 管理的范围

　　B 一名上级领导者直接领导的下级人员的人数

　　C 各层领导管理下级人员的人数总和

　　D 管理的权限

　　答案　B

　　11、有效管理幅度取决于（ ）。

　　A 以下三点

　　B 干部的素质

　　C 职务的性质

　　D 组织机构健全与否

　　答案　A

　　12、管理幅度与管理层次（ ）。

　　A 成正比例

　　B 成反比例

　　C 幅度增大就意味着层次增加

　　D 幅度增大就应该层次增加

　　答案　B

　　13、企业管理要（ ）。

　　A 重责轻权

　　B 重权轻责

　　C 责权利相结合

　　D 重权轻利

　　答案　C

　　14、责权利相结合原则，要求（ ）。

　　A 建立岗位责任制

　　B 责任和权力要一致

　　C 责任制必须同相应的经济利益结合起来

　　D 以上三点

　　答案　D

　　15、企业决策权需要（ ）。

　　A 集中化

　　B 集中化和分散化

　　C 分散化

　　D 注重集中，轻视分散

　　答案　B

　　16、企业管理，要有（ ）。

　　A 一定程度的集权

　　B 一定程度的分权

　　C 一定程度的集权和一定程度的分权

　　D 集权和分权，但不要平衡

　　答案　C

　　17、集权与分权，（ ）。

　　A 各有利弊

　　B 集权好于分权

　　C 分权好于集权

　　D 集权与分权相互独立

　　答案　A

　　18、企业集权和分权程度，取决于（ ）。

　　A 企业规模

　　B 生产技术特点

　　C 管理专业工作的性质

　　D 以上三点

　　答案　D

　　19、为了是企业的管理机构正常运转，首先需要（ ）。

　　A 统一性

　　B 合理性

　　C 稳定性

　　D 适应性

　　答案　C

　　20、为了适应变化的新情况，企业的管理机构必须具有（ ）。

　　A 稳定性

　　B 适应性

　　C 统一性

　　D 合理性

　　答案　B

　　21、精简机构，（ ）。

　　A 以完成任务为前提

　　B 以符合机构数量为前提

　　C 以削减机构数量为前提

　　D 以符合领导意图为前提

　　答案　A

　　22、精简机构，关键在于（ ）。

　　A 领导

　　B 加强人员培训和改进管理工作

　　C 干部数量

　　D 上级要求

　　答案　B

　　23、职能制结构，强调（ ）。

　　A 直线制

　　B 职能制

　　C 直线――参谋制

　　D 参谋制

　　答案　C

　　24、职能制结构，（ ）。

　　A 使企业管理权力高度集中

　　B 使企业管理权力分散

　　C 使专业分工不能变为现实

　　D 不易产生分散主义

　　答案　A

　　25、事业部是依据（ ）。

　　A 企业发展划分来的

　　B 企业现有机构划分来的

　　C 上级规定划分来的

　　D 企业经营的各种事业划分来的

　　答案　D

　　26、事业部制，有利于（ ）。

　　A 建立衡量管理人员成绩和效率的标准

　　B 发挥各自的积极性

　　C 培养全面的管理人才

　　D 以上三点

　　答案　D

　　27、事业部制组织结构，适合于（ ）。

　　A 中型企业

　　B 大型企业

　　C 小型企业

　　D 中、小型企业

　　答案　B

　　28、模拟分权制，是（ ）。

　　A 事业部制

　　B 职能制

　　C 参照事业部制的变形结构

　　D 直线制

　　答案　C

　　29、模拟分权制，（ ）。

　　A 拥有自己的职能机构

　　B 可以独立核算，自负盈亏

　　C 职责明确，组织经济性和决策的有效性强

　　D 不被广泛采用

　　答案　A

　　30、矩阵制组织结构，（ ）。

　　A 是按职能划分的纵向管理系统

　　B 是按工程（或规划项目、产品）划分的横向管理系统

　　C AB兼有

　　D 是直线――参谋制的对立系统

　　答案　C

　　31、矩阵制组织结构，（ ）。

　　A 灵活性、适宜性差

　　B 不利于协作配合

　　C 不利于工作人员的整体观念的加强

　　D 用人较多，机构比较臃肿

　　答案　D

　　（答案仅供参考）
