  • 09年自考“企业管理概论”练习题（3）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、根据各种因素对企业业绩影响程度的不同，企业所处的外部环境一般可分为（ ）

　　A 一般环境和任务环境

　　B 政治环境和经济环境

　　C 社会环境和文化环境

　　D 经济环境和技术环境

　　参考答案：A

　　2、对企业组织目标的实现有直接影响的外部环境因素属于 （ ）

　　A 政治环境

　　B 社会环境

　　C 任务环境

　　D 技术环境

　　参考答案：　C

　　3、战略决策与计划有客观的依据，反映事物的发展趋势，符合客观规律的要求。这是战略决策（ ）

　　A 科学性的要求

　　B 稳定性的要求

　　C 及时性的要求

　　D 效益性的要求

　　参考答案：A

　　4、决策方案要以尽可能小的代价，取得尽可能好的经济效益。这是战略决策 （ ）

　　A 科学性的要求

　　B 稳定性的要求

　　C 及时性的要求

　　D 效益性的要求

　　参考答案：　D

　　5、企业所在的国家或地区人们的处事态度、价值取向、教育程度、风俗习惯等构成的环境因素，在企业一般环境因素中属于 （ ）

　　A 政治环境因素

　　B 社会环境因素

　　C 经济环境因素

　　D 技术环境因素

　　参考答案：　B

　　6、一般从资金来源、人员供给、市场需求等方面影响企业的投入和产出，从而制约管理活动的因素，在企业一般环境因素中属于 （ ）

　　A 政治环境因素

　　B 社会环境因素

　　C 经济环境因素

　　D 技术环境因素

　　参考答案：　C

　　7、政府政策是指政府部门制定的包括经济、财政与税收政策以及各种重大的管理规定和法规。这在企业外部环境因素中属于 （ ）

　　A 政治环境因素

　　B 社会环境因素

　　C 技术环境因素

　　D 任务环境因素

　　参考答案：　D

　　8、资源供应的保证程度，在企业外部环境因素中属于 （ ）

　　A 政治环境因素

　　B 社会环境因素

　　C 经济环境因素

　　D 任务环境因素

　　参考答案：　D

　　9、企业外部环境重点调研的内容有（ ）

　　A 政治环境分析

　　B 社会环境分析

　　C 市场需求分析

　　D 技术环境分析

　　参考答案：　C

　　10、研究本企业的总体市场和各个细分市场在需求上的特点、发展趋势和影响因素，这在企业外部环境调研中属于（ ）

　　A 政治环境分析

　　B 社会环境分析

　　C 市场需求分析

　　D 经济环境分析

　　参考答案：　C

　　11、企业外部环境重点调研的内容有（ ）

　　A 政治环境分析

　　B 社会环境分析

　　C 竞争状况分析

　　D 经济环境分析

　　参考答案：　C

　　12、对市场商品供给状况的调查研究，这在企业外部环境调研中属于（ ）

　　A 政治环境分析

　　B 社会环境分析

　　C 竞争状况分析

　　D 经济环境分析

　　参考答案：　C

　　13、对构成企业内部生产经营过程的各种要素，即企业所拥有的客观物质条件和工作情况的分析，在企业环境分析中属于 （ ）

　　A 企业外部环境分析

　　B 企业经营实力分析

　　C 企业内部条件分析

　　D 企业竞争实力分析

　　参考答案：　C

　　14、为了能准确全面地弄清企业自身的优势和劣势、长处和短处，为内部挖掘潜力指明方向，创造条件。企业应重视 （ ）

　　A 企业外部环境分析

　　B 企业经营实力分析

　　C 企业内部条件分析

　　D 企业竞争实力分析

　　参考答案：　C

　　15、为全面了解企业管理水平是高还是低，各级管理者的管理知识、管理智能、管理技术是强还是弱，就必须要进行 （ ）

　　A 企业职工素质分析

　　B 企业管理素质分析

　　C 企业技术素质分析

　　D 企业组织结构分析

　　参考答案：　B

　　16、为了解企业生产过程组织和劳动组织是否适应市场的需求，能力结构与市场需求结构是否相适应，就必须要进行（ ）

　　A 企业生产条件分析

　　B 企业管理素质分析

　　C 企业技术素质分析

　　D 企业组织结构分析

　　参考答案：A

　　17、对产品竞争能力、技术开发能力、生产能力、市场营销能力和资金运用能力的分析，这属于 （ ）

　　A 企业外部环境分析

　　B 企业经营实力分析

　　C 企业一般情况分析

　　D 企业竞争实力分析

　　参考答案：　B

　　18、对企业所销售的产品的市场地位、收益性、成长性、竞争性等方面进行的分析。 这属于（ ）

　　A 市场营销能力分析

　　B 产品竞争能力分析

　　C 资金运用能力分析

　　D 生产能力的分析

　　参考答案：　B

　　19、将企业外部环境和企业内部条件的各种因素结合起来进行的分析。这是指（ ）

　　A 企业外部环境分析

　　B 企业经营实力分析

　　C 企业内部条件分析

　　D 企业战略综合分析

　　参考答案：　D

　　20、为全面了解企业发展中的机会和风险、优势和劣势，企业在战略环境的分析中必须要进行（ ）

　　A 企业外部环境分析

　　B 企业经营实力分析

　　C 企业内部条件分析

　　D 企业战略综合分析

　　参考答案：　D

　　21、企业内外情况对照分析法属于（ ）

　　A 外部环境分析方法

　　B 经营实力分析方法

　　C 内部条件分析方法

　　D 战略综合分析方法

　　参考答案：　D

　　22、为全面了解企业发展中的机会和风险、优势和劣势，企业在战略环境的分析中适于采用（ ）

　　A 波斯顿矩阵分析法

　　B 企业内外情况对照分析法

　　C 麦肯西矩阵分析法

　　D 经营实力分析法

　　参考答案：　B

　　23、将需求增长率和相对市场占有率作为衡量标准并形成矩阵图形，然后对企业的经营领域进行分析和评价的综合方法是（ ）

　　A 波斯顿矩阵分析法

　　B 企业内外情况对照分析法

　　C 麦肯西矩阵分析法

　　D 经营实力分析法

　　参考答案：A

　　24、在波斯顿矩阵分析法中，应严格控制投资，维持现有规模，设法获取尽可能多的利润的经营领域是（ ）

　　A "明星"区

　　B "现金牛"区

　　C "野猫"区

　　D "瘦狗"区

　　参考答案：　B

　　25、以战略经营领域的吸引力和企业的竞争地位两个综合性指标进行组合，形成矩阵，进行分析的综合性方法是（ ）

　　A 波斯顿矩阵分析法

　　B 企业内外情况对照分析法

　　C 麦肯西矩阵分析法

　　D 经营实力分析法

　　参考答案：　C

　　26、在麦肯西矩阵分析法中，应重点投资的经营领域是（ ）

　　A "明星"区

　　B "现金牛"区

　　C "野猫"区

　　D "瘦狗"区

　　参考答案：A

　　（答案仅供参考）
