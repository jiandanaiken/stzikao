  • 09年自考“企业管理概论”综合题（3）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、企业组织狭义地理解是企业组织的（ ），即企业组织结构。

　　A 核心

　　B 空间表现形式

　　C 内部关系

　　D 领导关系

　　答案　B

　　2、企业组织的广义理解，指企业财产机器社会化大生产的组织状态，具有（ ）的含义。

　　A 企业文化

　　B 企业管理

　　C 企业制度

　　D 企业系统

　　答案　C

　　3、企业制度是以（ ）为基础和核心的企业组织管理制度。

　　A 责任制度

　　B 社会制度

　　C 产权制度

　　D 教育制度

　　答案　C

　　4、从广义的组织形式角度分析，建立组织制度的意义有（ ）等。

　　A 优化企业的资源配置

　　B 提高企业劳动效率

　　C 促进分工与协作

　　D 加强党的领导

　　答案　A

　　5、从广义的组织形式角度分析，建立组织制度的意义有（ ）等。

　　A 提高企业劳动效率

　　B 明确企业的产权关系

　　C 促进分工与协作

　　D 加强党的领导

　　答案　B

　　6、从狭义的组织形式角度分析，建立组织制度的意义有（ ）等。

　　A 促进分工与协作

　　B 加强党的领导

　　C 明确企业产权关系

　　D 优化企业的资源配置

　　答案　A

　　7、从狭义的组织形式角度分析，建立组织制度的意义有（ ）等。

　　A 构成企业的骨骼系统

　　B 加强党的领导

　　C 明确企业产权关系

　　D 优化企业的资源配置

　　答案　A

　　8、个人业主制企业规模小，（ ）。

　　A 经营权与所有权分离

　　B 委托他人经营

　　C 经营权与所有权一体化

　　D 开业歇业手续复杂

　　答案　C

　　9、个人业主制企业规模小，（ ）。

　　A 经营权与所有权分离

　　B 委托他人经营

　　C 开业歇业手续复杂

　　D 经营灵活，产权转让灵活

　　答案　D

　　10、个人业主制企业规模小，具有（ ）的弱点。

　　A 资金庞大

　　B 难以扩大规模

　　C 产权转让不灵活

　　D 开业歇业手续复杂

　　答案　B

　　11、两个以上的个人共同出资，通过签订协议而量和经营的企业，称为（ ）。

　　A 有限责任公司

　　B 股份有限公司

　　C 合资企业

　　D 合伙制企业

　　答案　D

　　12、合伙制企业的特点首先表现在（ ）。

　　A 有限责任

　　B 规模较大

　　C 无限连带责任

　　D 承包经营

　　答案　C

　　13、合伙制企业是商品经济进一步发展的结果，但它仍有一定的缺点， 即（ ）。

　　A 资金数量很小

　　B 容易降低决策效率

　　C 权责不明确

　　D 企业信用程度低

　　答案　B

　　14、公司制企业是（ ）的组织形式。

　　A 个体经营

　　B 联合经营

　　C 承包经营

　　D 租赁经营

　　答案　B

　　15、公司制企业的特点主要有（ ）等。

　　A 经营权与所有权一体化

　　B 委托他人经营

　　C 开业歇业手续复杂

　　D 公司制企业是法人

　　答案　D

　　16、公司制企业的特点主要有（ ）等。

　　A 资金较小

　　B 实行有限责任制度

　　C 产权转让不灵活

　　D 开业歇业手续复杂

　　答案　B

　　17、同一行业中生产同类产品或同类部件的工厂联合组成的公司，称为（ ）。

　　A 有限责任公司

　　B 股份有限公司

　　C 合资企业

　　D 专业公司

　　答案　D

　　18、生产同类产品的专业公司的优点表现在有利于（ ）。

　　A 开发多种产品

　　B 扩大规模较大

　　C 内部组织技术交流

　　D 承包经营

　　答案　C

　　19、由若干个为其他企业提供技术服务的同类工艺工厂组成的专业公司 其优点是（ ）。

　　A 有利于开发产品

　　B 提高全社会的专业化水平，提高效益

　　C 有利于产品分工

　　D 提高企业信用程度低

　　答案　B

　　20、由若干彼此存在一定生产技术联系的，不同行业或生产不同类产品的工厂联合组成的企业，称为（ ）。

　　A 联合公司

　　B 股份有限公司

　　C 合资企业

　　D 合伙制企业

　　答案　A

　　21、以大中企业为依托，由若干具有独立法人资格、相互之间具有一定经济技术联系的企业结合而成的经济联合体，称为（ ）。

　　A 集团公司

　　B 股份公司

　　C 企业联合体

　　D 合资企业

　　答案　C

　　22、我国国有企业传统企业制度存在的弊端是（ ）。

　　A 政企职责不分

　　B 现代化水平低

　　C 政企分开

　　D 组织关系复杂

　　答案　B

　　23、我国国有企业改革的方向是（ ）。

　　A 建立现代企业制度

　　B 实行完全承包

　　C 建立合理的分配制度

　　D 利改税

　　答案　D

　　24、现代企业是以（ ）为核心的内容的全新企业制度。

　　A 企业法人财产

　　B 私人财产

　　C 人力资源

　　D 科学管理制度

　　答案　A

　　25、现代企业制度的是以（ ）为主要形式的企业制度。

　　A 合伙制

　　B 租赁制

　　C 公司制

　　D 承包制

　　答案　C

　　（答案仅供参考）
