　　1、在价值工程活动程序中，功能定义主要回答的问题是（ ）

　　A 这是干什么用的

　　B 它的价值是多少

　　C 有无新方案

　　D 新方案的成本多少

　　参考答案：A

　　2、在价值工程活动程序中，功能评价主要回答的问题是（ ）

　　A 这是干什么用的

　　B 它的价值多少

　　C 有无新方案

　　D 新方案的成本多少

　　参考答案：　B

　　3、将产品的各零件按其成本高低依次排列，绘制费用累积分配图，从中选择占成本比重大、占零件总数比重较小者作为价值工程的对象。这种选择价值工程对象的方法是（ ）

　　A 用户评分法

　　B 产品寿命周期法

　　C $$A%%B^^C法

　　D 百分比法

　　参考答案：　C

　　4、通过用户对产品各项性能的重要程度进行评分，从中找出价值工程对象的方法是指（ ）

　　A 用户评分法

　　B 产品寿命周期法

　　C $$A%%B^^C法

　　D 百分比法

　　参考答案：A

　　5、可作为价值工程活动的依据、标准、对比对象的是（ ）

　　A 情报资料

　　B 选择对象的方法

　　C 功能评价方法

　　D 组织领导

　　参考答案：A

　　6、价值工程对象确定之后，第二步的工作就是要围绕所选择的对象（ ）

　　A 进行功能整理

　　B 进行功能评价

　　C 拟定改进方案

　　D 收集情报资料

　　参考答案：　D

　　7、明确产品的必要功能，搞清产品功能之间的关系，确定产品合理的功能结构。这是（ ）

　　A 产品功能分析的目的

　　B 产品功能评价的目的

　　C 开展价值工程的目的

　　D 确定改进方案的目的

　　参考答案：A

　　8、对产品功能的要求，主要考虑三个方面的因素，即社会环境的要求、企业自身的要求，以及（ ）

　　A 技术的要求

　　B 用户的要求

　　C 产品的要求

　　D 经济的要求

　　参考答案：　B

　　9、对产品及其零部件所应具备的功能，用简明准确的语言进行描述。这是指对产品的（ ）

　　A 功能评价

　　B 功能整理

　　C 功能定义

　　D 功能分类

　　参考答案：　C

　　10、在功能定义中，功能的表达要尽量（ ）

　　A 抽象化

　　B 具体化

　　C 定性化

　　D 简单化

　　参考答案：A

　　11、在功能分类中，与使用者的精神感觉、主观意识有关的功能是（ ）

　　A 使用功能

　　B 美学功能

　　C 基本功能

　　D 不必要功能

　　参考答案：　B

　　12、在功能分类中，按功能的重要程度划分有（ ）

　　A 基本功能和辅助功能

　　B 使用功能和美学功能

　　C 精神功能和物质功能

　　D 必要功能和不必要功能

　　参考答案：A

　　13、在功能定义完成之后，从系统分析的角度明确功能之间的相互关系，最终形成功能系统图。这是指（ ）

　　A 功能评价

　　B 功能整理

　　C 功能定义

　　D 功能分类

　　参考答案：　B

　　14、在功能之间的相互关系中，上下关系是指（ ）

　　A 上位功能是目的，下位功能是手段

　　B 下位功能是目的，上位功能是手段

　　C 上位功能与下位功能是并列关系

　　D 上位功能与下位功能无相关关系

　　参考答案：A

　　15、定量的探讨功能的价值，找出价值功能较低者，明确需要改进的具体功能范围。这是指（ ）

　　A 功能评价

　　B 功能整理

　　C 功能定义

　　D 功能分类

　　参考答案：A

　　16、通过功能评价，应作为价值工程改进重点的是（ ）

　　A 价值系数高于1的零件

　　B 价值系数低于1，成本比重小的零件

　　C 价值系数等于1的零件

　　D 价值系数低于1，成本比重大的零件

　　参考答案：　D

　　17、在确定功能评价值的方法中，根据功能重要程度用百分制、十分制或五分制打分，按分数分配成本的方法。这是指（ ）

　　A 经验估计法

　　B 直接打分法

　　C 强制确定法

　　D 倍数确定法

　　参考答案：　B

　　18、根据零件的重要程度，采取一对一对比打分法打分，然后把各零件的分数占总分的比重作为各自的评价系数，并把目标成本按评价系数分配，求出各零件的功能评价值。这是指（ ）

　　A 经验估计法

　　B 直接打分法

　　C 强制确定法

　　D 倍数确定法

　　参考答案：　C

　　19、对各种改进方案，分析其优缺点，进行评价，并把技术和经济上都较优越的方案进一步完善，最后选出最佳方案。这是指（ ）

　　A 功能评价

　　B 分析与评价方案

　　C 试验与定案

　　D 选择价值工程对象

　　参考答案：　B

　　20、根据各种方案能够达到各种功能要求的程度，按五分制、十分制或百分制打分，然后算出每个方案达到功能要求的总分。比较各方案的总分，确定最优方案。这是指（ ）

　　A 优缺点列举法

　　B 直接打分法

　　C 加权评分法

　　D 组合法

　　参考答案：　B

　　21、把产品分解为几个重要组成部分，分别制定改进方案，然后从各组成部分的不同方案中进行选择，最后，把选择出的各个局部方案组成整体方案。这是指（ ）

　　A 优缺点列举法

　　B 直接打分法

　　C 加权评分法

　　D 组合法

　　参考答案：　D

　　（答案仅供参考）
