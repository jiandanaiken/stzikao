  • 09年自考“企业管理概论”练习题（15）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、市场营销工作应以（ ）为中心。

　　A 企业

　　B 地区

　　C 产品

　　D 消费者

　　参考答案：　D

　　2、人们的主观需要引起人们主观的购买动机，即需要对购买的物品进行（ ）。

　　A 品质的选择

　　B 数量的选择

　　C 感情的选择

　　D 品种的选择

　　参考答案：　C

　　3、消费者有购买动机到购买行为发生要经历（ ）心理过程。

　　A 五个

　　B 三个

　　C 四个

　　D 两个

　　参考答案：A

　　4、消费者购买规律决定了其购买行为分为五个阶段，它的本源是（ ）。

　　A 消费者动机

　　B 消费者行为

　　C 消费者需要

　　D 消费者情感

　　参考答案：　C

　　5、实现销售是企业再生产过程的完成，对消费者来说是（ ）。

　　A 购买行为的发生

　　B 购买行为的完结

　　C 购买动机的产生

　　D 需要的产生

　　参考答案：A

　　6、完成商品销售是企业商品销售五个阶段的（ ）。

　　A 过程

　　B 经历

　　C 目的

　　D 手段

　　参考答案：　C

　　7、企业选择和确定自己的目标市场应符合一定的条件，其中包括（ ）。

　　A 该市场是垄断的

　　B 企业在此方面实力欠缺

　　C 该市场具有一定的购买能力

　　D 该市场竞争激烈

　　参考答案：　C

　　8、企业选择和确定自己的目标市场应符合一定的条件，其中包括（ ）。

　　A 该市场购买力不足

　　B 企业资金雄厚

　　C 企业实力欠缺

　　D 企业具有一定优势，有能力进入该市场

　　参考答案：　D

　　9、企业确定目标市场要根据企业实力和资源状况确定（ ）的选择。

　　A 目标市场范围

　　B 目标市场营销

　　C 目标市场份额

　　D 目标市场性质

　　参考答案：A

　　10、企业确定目标市场要处理好几方面的关系，其中包括（ ）。

　　A 顾客与顾客的关系

　　B 企业与企业的关系

　　C 当前市场与长远市场的关系

　　D 区域市场与全国市场的关系

　　参考答案：　C

　　11、企业确定目标市场要处理好几方面的关系，其中包括（ ）。

　　A 消费品市场与生产资料市场的关系

　　B 企业与企业的关系

　　C 高档市场与低档市场的关系

　　D 本地市场与外部市场的关系

　　参考答案：　D

　　12、企业确定目标市场要处理好几方面的关系，其中包括（ ）。

　　A 重点市场与一般市场的关系

　　B 边缘市场与中心市场关系

　　C 现货市场与期货市场的关系

　　D 企业与企业的关系

　　参考答案：A

　　13、在企业市场定位中，采取同竞争对手相同的市场定位策略称为（ ）。

　　A 避强策略

　　B 雷同策略

　　C 超强策略

　　D 差异化策略

　　参考答案：　B

　　14、避开强大对手的锋芒，选择那些不为强大对手注意的目标市场的策略称为（ ）。

　　A 避强策略

　　B 雷同策略

　　C 超强策略

　　D 差异化策略

　　参考答案：A

　　15、不仅选择与强手相同的市场定位，而且在同一定位市场上要处于更强强有力的竞争地位，这种策略是（ ）。

　　A 避强策略

　　B 雷同策略

　　C 超强策略

　　D 差异化策略

　　参考答案：　C

　　16、在目标市场上针对竞争对手产品在该市场的地位和顾客对产品的态度，有目的地树立本企业形象，确定目标市场上的位置，这种策略称为（ ）。

　　A 细分市场策略

　　B 产品竞争策略

　　C 目标市场选择策略

　　D 产品市场定位策略

　　参考答案：　D

　　17、企业进入市场的空间策略中有一种是先进入本地或附近地区市场，然后逐步推广到外地市场的策略称为（ ）。

　　A 先远后近策略

　　B 齐头并进策略

　　C 先近后远策略

　　D 整体推进策略

　　参考答案：　C

　　18、企业进入市场的空间策略中有一种是先进入外地或较远地区市场，然后进入本地市场的策略是（ ）。

　　A 先远后近策略

　　B 齐头并进策略

　　C 先近后远策略

　　D 整体推进策略

　　参考答案：A

　　19、企业开发的新产品具有较强的实力，在本地和外地同时进入市场的策略是（ ）。

　　A 先远后近策略

　　B 齐头并进策略

　　C 先近后远策略

　　D 整体推进策略

　　参考答案：　B

　　20、企业在选择进入目标市场的时间时，在掌握对手实力较强，也有新产品即将投放市场时，应采取（ ）的策略。

　　A 提早进入市场

　　B 适中进入市场

　　C 较晚进入市场

　　D 退出市场

　　参考答案：A

　　21、企业在选择进入目标市场的时间时，根据顾客购买力的情况选择投入时机是（ ）的策略。

　　A 提早进入市场

　　B 适中进入市场

　　C 较晚进入市场

　　D 退出市场

　　参考答案：　B

　　22、企业在选择进入目标市场的时间时，在掌握对手迟迟不能推出新产品本企业有较多老产品销路较好，应采取（ ）的策略。

　　A 提早进入市场

　　B 适中进入市场

　　C 较晚进入市场

　　D 退出市场

　　参考答案：　C

　　23、市场细分的根据是众多用户或众多消费者的（ ）。

　　A 不同地理位置

　　B 不同需求特点

　　C 不同性别

　　D 不同收入水平

　　参考答案：　B

　　24、市场细分就是根据消费者的不同需求特点，将企业某产品的总体市场（ ）。

　　A 进行分割

　　B 进行估算

　　C 划分为若干个消费者群

　　D 和其他市场综合

　　参考答案：　C

　　25、市场细分实际上是企业根据众多消费者不同需求特点，进行（ ）

　　A 市场的整合过程

　　B 市场的调研过程

　　C 市场的筛选过程

　　D 细小市场的分类过程

　　参考答案：　D

　　26、市场营销对企业具有重要作用，其中（ ）是作用之一。

　　A 将不适销的产品推出去

　　B 掌握市场分布情况，发现市场缝隙；

　　C 引发顾客抢购的心理

　　D 为企业带来高额利润

　　参考答案：　B

　　27、市场营销对企业具有重要作用，其中（ ）是作用之一。

　　A 便于确定重点目标市场

　　B 便于推销积压产品

　　C 便于获得高额利润

　　D 便于垄断整个市场

　　参考答案：A

　　28、市场营销对企业具有重要作用，其中（ ）是作用之一。

　　A 了解市场风险，便于退出市场

　　B 了解同业动向

　　C 掌握顾客需求特点，正确进行产品开发河市场开发

　　D 有效进行资产重组

　　参考答案：　C

　　29、市场细分按照产品用途不同，将市场分为两大类总体市场（ ）

　　A 耐用商品市场和易耗商品市场

　　B 高档商品市场和普通商品市场

　　C 个人消费品市场和团体消费品市场

　　D 消费品市场和生产资料产品市场

　　参考答案：　D

　　30、以人口、地理区划、购买心理和购买行为等为标志进行细分的市场是（ ）。

　　A 生产资料产品市场

　　B 消费品市场

　　C 国际市场

　　D 工业品市场

　　参考答案：　B

　　31、按照企业规模、客户使用要求、产业和行业特点等进行市场细分的市场是（ ）。

　　A 生产资料产品市场

　　B 生活资料产品市场

　　C 国际市场

　　D 国内市场

　　参考答案：A

　　32、市场细分的要求包括（ ）和其他四点要求。

　　A 不断细分

　　B 标志统一不变

　　C 可衡量性

　　D 利润最大化

　　参考答案：　C

　　33、市场细分的要求包括（ ）和其他四点要求。

　　A 市场开发的效益性

　　B 不断细分

　　C 瞄准新市场

　　D 标志统一不变

　　参考答案：A

　　34、市场细分的要求包括（ ）和其他四点要求。

　　A 标志统一不变

　　B 划分标志的动态性

　　C 不断细分

　　D 不可衡量
