  • 09年自考“企业管理概论”练习题（4）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、企业在推动社会进步和发展国民经济中所担当的责任。这是指 （ ）

　　A 企业战略目标

　　B 企业战略方针

　　C 企业使命

　　D 企业宗旨

　　参考答案：　C

　　2、企业战略目标的制定通常从 （ ）

　　A 确定战略方针开始

　　B 确定企业使命开始

　　C 确定战略政策开始

　　D 确定战略阶段开始

　　参考答案：　B

　　3、企业在创造物质财富和精神财富的实践中所表现出来的世界观和方法论，表明企业对它们承担的使命和所起的作用，以及如何起作用的总体看法。这一概念是指 （ ）

　　A 企业战略目标

　　B 企业战略方针

　　C 企业经营哲学

　　D 企业宗旨

　　参考答案：　C

　　4、将企业经营哲学具体化，具体表明企业从事什么生产经营活动，达到什么目的和对社会起什么作用。这一概念是指 （ ）

　　A 企业战略目标

　　B 企业战略方针

　　C 企业经营哲学

　　D 企业宗旨

　　参考答案：　D

　　5、将企业使命和宗旨具体化和定量化，把企业经营的目的转化为多方面的可以量化的具体指标，反映企业经过一定时间内应达到的经营水平。这一概念是指 （ ）

　　A 企业战略目标

　　B 企业战略方针

　　C 企业经营哲学

　　D 企业宗旨

　　参考答案：A

　　6、战略目标必须建立在全面分析企业内部条件和外部环境的基础上，不能脱离实际，必须适中、可行。这是战略目标 （ ）

　　A 可接受性的要求

　　B 可检验性的要求

　　C 可实现性的要求

　　D 可挑战性的要求

　　参考答案：　C

　　7、在战略目标的组成中，表明企业资源利用程度的指标是指 （ ）

　　A 发展性目标

　　B 收益性目标

　　C 有效性目标

　　D 社会性目标

　　参考答案：　C

　　8、在战略目标的组成中，表明企业承担社会责任对社会做出贡献的指标是指 （ ）

　　A 发展性目标

　　B 收益性目标

　　C 有效性目标

　　D 社会性目标

　　参考答案：　D

　　9、旨在现有规模和实力的基础上，向更高的目标推进，以使企业经营规模或经营领域不断发展的战略。是指 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 保守性战略

　　参考答案：A

　　10、处于成长期的企业或者在经营领域中处于优势地位的企业，一般适于采用的经营战略是 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 保守性战略

　　参考答案：A

　　11、旨在内外环境的约束下，以安全经营为宗旨，将企业的经营状况和水平维持在现有基础上的战略。是指 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 进攻型战略

　　参考答案：　B

　　12、在经营领域中没有突出的优势地位，受各种资源条件制约程度较大的企业，一般适于采用的经营战略是 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 进攻型战略

　　参考答案：　B

　　13、旨在企业在内外环境的制约下，在现有的经营状态或经营领域中退却的战略。是指 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 进攻型战略

　　参考答案：　C

　　14、市场发生明显变化，由于经营领域中的失误导致失败，不得不收缩的企业，一般适于采用的经营战略是 （ ）

　　A 发展型战略

　　B 稳定型战略

　　C 紧缩型战略

　　D 进攻型战略

　　参考答案：　C

　　15、旨在为寻求使自己的生产成本达到本行业最低，从而拥有竞争优势的战略。是指 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 集中化战略

　　参考答案：A

　　16、产品的生产规模能够达到规模经济的要求，拥有先进的技术和专用设备，有持续的投资和良好的融资能力的企业，一般适于采用的经营战略是 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 集中化战略

　　参考答案：A

　　17、旨在通过在产品的性能、外观、服务和营销策略等方面，寻求与竞争对手的差异化，求取竞争优势的战略。是指 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 集中化战略

　　参考答案：　B

　　18、有较强的产品研究开发和市场营销能力，能够在产品设计、工艺技术、营销上不断发挥创造性；在产品质量、技术，以及营销和服务等方面有着良好的声誉的企业，一般适于采用的经营战略是 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 集中化战略

　　参考答案：　B

　　19、旨在集中力量专门为某个或少数几个特殊的细分市场服务，从而寻求在局部市场上拥有竞争优势的战略。是指 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 多角经营战略

　　参考答案：　C

　　20、中小企业由于无法在全方位的领域同实力强大的大企业展开竞争，一般适于采用的经营战略是 （ ）

　　A 低成本战略

　　B 差异化战略

　　C 重点化战略

　　D 多角经营战略

　　参考答案：　C

　　21、按不同专业职能对总体战略进行落实和具体化的战略。即在总体战略指导下，为企业各部门、所属各单位制定的战略。这是指 （ ）

　　A 总体战略

　　B 产品战略

　　C 职能战略

　　D 技术战略

　　参考答案：　C

　　22、规定着企业某一方向或某一领域的努力方向，并服从于企业发展的总方向。这是职能战略的 （ ）

　　A 从属性特点

　　B 单一性特点

　　C 专业性特点

　　D 针对性特点

　　参考答案：A

　　23、产品的市场容量大，需求前景很好，产品利润率高，企业的经营实力又强，即开发和生产能力强的企业，一般适于采用的市场战略是（ ）

　　A 扩张型市场战略

　　B 维持型市场战略

　　C 撤退型市场战略

　　D 稳定型市场战略

　　参考答案：A

　　24、产品容量有限，需求饱和，市场竞争又十分激烈，企业的经营实力也属中等的企业，一般适于采用的市场战略是 （ ）

　　A 扩张型市场战略

　　B 维持型市场战略

　　C 撤退型市场战略

　　D 进攻型市场战略

　　参考答案：　B

　　25、根据经营战略决策方案有关目标的要求，对方案实施所需的各种资源，从时间和空间上所作出的统筹安排。这一概念是指 （ ）

　　A 经营方针

　　B 经营目标

　　C 经营计划

　　D 经营对策

　　参考答案：　C

　　26、经营计划的制定要建立在对社会和市场的全面了解的基础之上，计划的制定要同市场和用户发生直接联系。这是经营计划 （ ）

　　A 决策性的特点

　　B 效益性的特点

　　C 开放性的特点

　　D 灵活性的特点

　　参考答案：　C

　　27、通过经营计划可以使各单位、各部门的具体目标相互协调，以保证企业总体目标的实现。这是经营计划具有的 （ ）

　　A 协调作用

　　B 分配作用

　　C 保证作用

　　D 激励作用

　　参考答案：A

　　28、根据总体目标要求和各单位各部门所承担的具体目标任务，对其所需资源进行合理的分配。这是经营计划具有的 （ ）

　　A 协调作用

　　B 分配作用

　　C 保证作用

　　D 激励作用

　　参考答案：　B

　　29、按经营计划的对象划分，经营计划有 （ ）

　　A 综合经营计划和单项经营计划

　　B 经营战略计划和经营战术计划

　　C 长期经营计划、中期经营计划和短期经营计划
