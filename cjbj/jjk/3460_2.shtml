　　35、市场细分的要求包括（ ）和其他四点要求。

　　A 标志统一不变

　　B 不断细分

　　C 利润最大化

　　D 划分特性的可衡量性

　　参考答案：　D

　　36、企业市场营销，是企业为满足市场需要，实现自身经营目标而开展的（ ）。

　　A 生产活动

　　B 调研活动

　　C 商务活动

　　D 服务活动

　　参考答案：　C

　　37、市场营销活动工作的性质是企业再生产过程中属于（ ）的工作。

　　A 流通领域

　　B 生产领域

　　C 服务领域

　　D 消费领域

　　参考答案：A

　　38、市场营销活动必须以（ ）为导向。

　　A 企业盈利

　　B 社会效益

　　C 政府要求

　　D 顾客的需求

　　参考答案：　D

　　39、在生产经营型管理的条件下，销售工作在企业生产经营活动的（ ）。

　　A 第二线

　　B 第一线

　　C 后方

　　D 外部

　　参考答案：　B

　　40、市场营销是属于开辟市场、占领市场和扩大市场的工作，它必须以（ ）为依据。

　　A 企业目标

　　B 地区利益

　　C 生产进度

　　D 市场

　　参考答案：　D

　　41、市场营销工作的作用在于解决（ ）的矛盾。

　　A 生产与消费

　　B 生产与供应的矛盾

　　C 企业与外部

　　D 生产与销售

　　参考答案：A

　　42、市场营销是联系（ ）的纽带。

　　A 企业部门之间

　　B 企业与企业之间

　　C 生产与消费

　　D 消费与消费

　　参考答案：　C

　　43、企业市场营销工作主要包括市场调查工作、目标市场选择工作、产品定价工作、销售渠道选择工作、产品储存和运输工作以及承接订货和销售服务 、（ ）等八项工作。

　　A 产品促销和产品开发工作

　　B 产品包装工作

　　C 产品销售工作

　　D 产品更新和促销工作

　　参考答案：A

　　44、企业市场营销工作主要包括产品开发工作、目标市场选择工作、产品定价工作、产品促销工作、产品储存和运输工作以及承接订货和销售服务、（ ）等八项工作。

　　A 市场细分

　　B 市场定位

　　C 市场调研和销售渠道选择工作

　　D 市场分析

　　参考答案：　C

　　45、企业市场营销工作的内容十分广泛，但不是所有的工作都属于营销工作，下列哪项工作是属于营销工作范畴？（ ）

　　A 企业重组

　　B 人力资源开发

　　C 承接订货和销售服务

　　D 社会公益活动

　　参考答案：　C

　　（答案仅供参考）
