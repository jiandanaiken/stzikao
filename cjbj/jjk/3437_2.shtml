资金

　　答案　C

　　38、完善企业的制度化管理应谋求（ ）的协调。

　　A 设备与人

　　B 组织之间

　　C 制度化与人性、制度化与活力

　　D 制度化与人情化

　　答案　C

　　39、企业技术业务培训工作的意义在于（ ）。

　　A 提升企业人力资源的水平

　　B 提高全员文化水平

　　C 实现精神文明建设目标

　　D避免内部竞争

　　答案　A

　　40、企业技术业务培训工作的意义在于（ ）。

　　A 提高劳动效率降低成本

　　B 减少生产事故

　　C 实现精神文明建设目标

　　D避免内部竞争

　　答案　A

　　41、企业技术业务培训工作的意义在于（ ）。

　　A 减少生产事故

　　B 增强团队精神提高组织综合力

　　C 实现精神文明建设目标

　　D避免内部竞争

　　答案　B

　　42、做好企业技术业务培训工作应（ ）。

　　A 建立职工教育机构

　　B 组织全员脱产学习

　　C 围绕政治学习

　　D避免内部竞争

　　答案　A

　　43、做好企业技术业务培训工作应（ ）。

　　A 聘请大学教授

　　B 根据企业需要和职工现状

　　C 根据国家教育规定

　　D根据其他企业要求

　　答案　B

　　（答案仅供参考）
