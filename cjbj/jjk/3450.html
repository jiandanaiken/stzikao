  • 09年自考“企业管理概论”练习题（5）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、对于企业中第一次应用或出现的新技术所开展的一系列活动包括创造学习、适应、消化、掌握、修改等过程。（ ）

　　A 价值工程

　　B 技术开发

　　C 工艺准备

　　D 生产准备

　　参考答案：　B

　　2、建立在基础研究和应用研究基础上的研究新成果，在技术开发类型中通常是指（ ）

　　A 小革新、小发明

　　B 局部革新

　　C 技术改造

　　D 技术创新和发明

　　参考答案：　D

　　3、运用新的科技成果，对产品的加工制造方法、技术和过程等进行的改进和革新，这在技术开发对象中是指（ ）

　　A 产品的开发

　　B 生产工艺的开发

　　C 原材料的开发

　　D 设备与工具的开发

　　参考答案：　B

　　4、工业企业技术开发中最主要的内容应是（ ）

　　A 产品的开发

　　B 生产工艺的开发

　　C 原材料的开发

　　D 设备与工具的开发

　　参考答案：A

　　5、要从基础研究搞起，科研难度大，时间长，耗费投资多，对科研人员的素质要求高，具有这种特点的技术开发途径是指（ ）

　　A 独创型

　　B 引进（转移）型

　　C 综合与延伸型

　　D 总结提高型

　　参考答案：A

　　6、从有利于企业保持技术上的领先地位的角度看，企业适于采用的技术开发途径是（ ）

　　A 独创型

　　B 引进（转移）型

　　C 综合与延伸型

　　D 总结提高型

　　参考答案：A

　　7、引进成套或关键技术、设备，由本企业工程技术人员掌握使用。这在引进（转移）型技术开发途径中式是指（ ）

　　A 移植

　　B 嫁接

　　C 插条

　　D 交配

　　参考答案：A

　　8、从外部引初步研究成果，在本企业进一步加以培植、最后形成产品。这在引进（转移）型技术开发途径中式是指（ ）

　　A 移植

　　B 嫁接

　　C 插条

　　D 交配

　　参考答案：　C

　　9、把两项或多项现有技术组合起来，由此创造和发明新的技术或新的产品。具有这种特点的技术开发途径是指（ ）

　　A 延伸型

　　B 引进型

　　C 综合型

　　D 总结提高型

　　参考答案：　C

　　10、对现有技术向技术的深度、强度、规模等方向的开发。具有这种特点的技术开发途径是指（ ）

　　A 延伸型

　　B 引进型

　　C 综合型

　　D 总结提高型

　　参考答案：A

　　11、以小革新、小建议、小发明等为主体的小改小革活动。这在技术开发途径中是指（ ）

　　A 独创型

　　B 引进（转移）型

　　C 综合与延伸型

　　D 总结提高型

　　参考答案：　D

　　12、通过生产实践经验的总结、提高来开发新技术。这在技术开发途径中是指（ ）

　　A 独创型

　　B 引进（转移）型

　　C 综合与延伸型

　　D 总结提高型

　　参考答案：　D

　　13、通常用来反映企业技术开发实力大小的重要标志是（ ）

　　A 技术开发装备

　　B 技术开发经费

　　C 技术开发场所

　　D 科技工作队伍

　　参考答案：　B

　　14、一支合格的科技队伍应满足两个基本要求，即要有一定的数量，又要有（ ）

　　A 技术开发装备

　　B 技术开发经费

　　C 技术开发场所

　　D 合理的队伍结构

　　参考答案：　D

　　15、科技队伍中各类专业人员的比例，这在科技队伍结构中是指（ ）

　　A 年龄结构

　　B 能级结构

　　C 职能结构

　　D 学科结构

　　参考答案：　D

　　16、企业中高、中、初级技术人员的比例，这在科技队伍结构中是指（ ）

　　A 年龄结构

　　B 能级结构

　　C 职能结构

　　D 学科结构

　　参考答案：　B

　　17、按技术的形态进行分类，可以把技术分为 （ ）

　　A 物质与智能形态的技术

　　B 传统技术与现代技术

　　C 代替技术与极限技术

　　D 综合与延伸技术

　　参考答案：A

　　18、按新、老技术相互关系进行分类，可以把技术分为（ ）

　　A 物质与智能形态的技术

　　B 传统技术与现代技术

　　C 代替技术与极限技术

　　D 综合与延伸技术

　　参考答案：　C

　　19、先进技术、中间技术与初级技术之间的比例关系。这在技术结构中是指（ ）

　　A 要素结构

　　B 层次结构

　　C 相关结构

　　D 序列结构

　　参考答案：　B

　　20、在产品开发过程中，从技术的先后程序来考察技术的结构，这在技术结构中是指（ ）

　　A 要素结构

　　B 层次结构

　　C 相关结构

　　D 序列结构

　　参考答案：　D

　　21、技术改造与基建项目中，生产技术设施同非生产设施之间的配套，这在技术配套关系中属于（ ）

　　A 技术要素内部的配套

　　B 技术要素之间的配套

　　C 技术要素与其它项目之间的配套

　　D.

　　参考答案：　C

　　22、对技术开发的课题选择与成果评价时，不仅要考虑对到当前本企业发展的作用，还要考虑到今后对本企业较长时间的影响。这一技术评价的原则是指（ ）

　　A 当前和长远需要相结合

　　B 局部与整体利益相结合

　　C 技术先进性与生产可行性相结合

　　D 技术先进性与经济合理性相结合

　　参考答案：A

　　23、新技术成果的效益分析，不仅要能为本企业带来效益，而且能为本行业、本地区、乃至整个国民经济的发展带来效益。这一技术评价的原则是指（ ）

　　A 当前和长远需要相结合

　　B 局部与整体利益相结合

　　C 技术先进性与生产可行性相结合

　　D 技术先进性与经济合理性相结合

　　参考答案：　B

　　24、从市场竞争能力、需要程度、销路等市场角度进行的技术评价是指对技术的（ ）

　　A 技术先进性的评价

　　B 经济价值的评价

　　C 社会价值的评价

　　D 技术适用性的评价

　　参考答案：　B

　　25、从技术的扩散效应、相关技术的匹配、实用程度、形成的技术优势等角度进行的技术评价是指对技术的（ ）

　　A 技术先进性的评价

　　B 经济价值的评价

　　C 社会价值的评价

　　D 技术适用性的评价

　　参考答案：　D

　　26、对技术开发课题的预测收益同开发该课题的预计费用进行对比，计算课题收益率。这在技术评价中属于（ ）

　　A 经济评价法

　　B 综合评价法

　　C 加权评分法

　　D 倍数评分法

　　参考答案：A

　　27、在定量分析的基础上，从系统的整体概念出发，将各种因素的得分，用加法计算总分，依据总分决定课题的优先顺序或应用的可能。这在技术评价中属于（ ）

　　A 经济评价法

　　B 综合评价法

　　C 加权评分法

　　D 倍数评分法

　　参考答案：　B

　　28、技术成果能否迅速转化为现实的生产力，取决于技术成果能否得到（ ）

　　A 推广和应用

　　B 技术先进性

　　C 技术适用性

　　D 技术创新性

　　参考答案：A

　　29、技术成果能否得到推广和应用，其重要条件之一是（ ）

　　A 技术价值性

　　B 技术创新性

　　C 技术成熟性

　　D 技术先进性

　　参考答案：　C

　　30、把同类型产品的主要参数，通过技术经济分析，按照使用要求及经济合理原则，进行选择排列，发展结构先进的品种，编成产品系列。这是指产品的（ ）

　　A 通用化

　　B 系列化

　　C 标准化

　　D 经济化

　　参考答案：　B

　　31、对于大量生产和广泛使用的产品和零部件，按照经济适用原则，将几种性能规格相似的统一为一种。这是指产品的（ ）

　　A 通用化

　　B 系列化

　　C 标准化

　　D 经济化

　　参考答案：A

　　32、对新开发的产品结构方案、组成整机的零部件，以及与之相关的新原理、新工艺等方面进行专题试验研究，为新产品的设计提供科学上的依据。这在产品设计的程序中属于（ ）

　　A 初步设计

　　B 工艺设计

　　C 技术设计

　　D 工作图设计

　　参考答案：A

　　33、提供试制和生产所需的全套图纸，提供试制、制造和使用所需的全部技术文件。这在产品设计的程序中属于（ ）

　　A 初步设计

　　B 工艺设计

　　C 技术设计

　　D 工作图设计

　　参考答案：　D

　　34、解决确定使用什么样的设备和工具，采用什么样的加工顺序和方法生产出产品的问题。这属于（ ）

　　A 技术设计

　　B 工艺准备

　　C 试制鉴定

　　D 设计准备

　　参考答案：　B
