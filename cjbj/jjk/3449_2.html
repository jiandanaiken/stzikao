　　D 实物量计划和价值计划

　　参考答案：A

　　30、按计划的作用划分，经营计划有 （ ）

　　A 综合经营计划和单项经营计划

　　B 经营战略计划和经营战术计划

　　C 长期经营计划、中期经营计划和短期经营计划

　　D 实物量计划和价值计划

　　参考答案：　B

　　31、规定企业为完成计划任务所需要的各类人员的数量、劳动生产率提高的水平、职工的工资水平等。这在经营计划内容中属于 （ ）

　　A 生产计划

　　B 销售计划

　　C 劳动工资计划

　　D 企业改造计划

　　参考答案：　C

　　32、规定企业生产的产品品种、数量、质量及生产进度。这在经营计划内容中属于 （ ）

　　A 生产计划

　　B 销售计划

　　C 劳动工资计划

　　D 企业改造计划

　　参考答案：A

　　33、经营计划编制中的关键步骤是 （ ）

　　A 调查研究，确定前提

　　B 统筹安排，确定目标

　　C 拟定方案，评价选择

　　D 综合平衡，编制计划

　　参考答案：　B

　　34、经营计划编制的前提是 （ ）

　　A 调查研究

　　B 确定目标

　　C 选择方案

　　D 综合平衡

　　参考答案：A

　　35、按照方针及目标的要求，对一切与之有关的部门、单位，提出进一步的要求，使之形成为一个系统，确保方针和目标的实现。这是指（ ）

　　A 方针展开

　　B 目标管理

　　C 计划管理

　　D 组织管理

　　参考答案：A

　　36、将经营目标细分为若干小指标，层层落实到有关部门、各单位，直到个人，变成具体目标，并形成一个目标体系。 这是指 （ ）

　　A 方针展开

　　B 目标管理

　　C 计划管理

　　D 组织管理

　　参考答案：　B

　　37、经营计划的调整是指 （ ）

　　A 对计划实施过程出现偏差的调整

　　B 对计划本身出现偏差的调整

　　C 对计划执行人员工作偏差的调整

　　D 是对原有经营计划的肯定

　　参考答案：A

　　38、将长期计划、中期计划和短期计划有机结合起来，根据计划实施过程中的变化，定期修订计划并逐期向前推移的方法。这是指（ ）

　　A 启用备用计划法

　　B 滚动计划法

　　C 甘特图法

　　D 网络计划法

　　参考答案：　B

　　39、有利于将组织的长期、中期和短期计划紧密的衔接起来，使计划更加富有弹性，提高了组织对环境的应变能力的计划方法是 （ ）

　　A 启用备用计划法

　　B 滚动计划法

　　C 甘特图法

　　D 网络计划法

　　参考答案：　B

　　（答案仅供参考）
