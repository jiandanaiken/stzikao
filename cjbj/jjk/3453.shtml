  • 09年自考“企业管理概论”练习题（8）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、可按生产技术特点、接受生产任务的方式、生产的连续程度、产品生产的重复程度和工作地的专业化程度进行划分的是（ ）。

　　A、生产类别

　　B、合理组织生产的形式

　　C、车间形式

　　D、流水线形式

　　参考答案：A

　　2、生产类型是（ ）。

　　A、生产类别

　　B、按不同标志划分的生产类别

　　C、大批生产

　　D、计划的选择形式

　　参考答案：　B

　　3、提取型生产方式是按照（ ）划分的。

　　A、生产技术特点

　　B、接受生产任务的方式

　　C、生产连续程度

　　D、成批生产的重复程度和工作地专业化程度

　　参考答案：A

　　4、合成型生产方式是按照（ ）划分的。

　　A、接受生产任务方式

　　B、生产技术特点

　　C、生产连续程度

　　D、生产的重复程度和工作地专业化程度

　　参考答案：　B

　　5、订货生产是按照（ ）划分的

　　A、生产技术特点

　　B、接受生产任务方式

　　C、生产连续程度

　　D、生产的重复程度和工作地专业化程度

　　参考答案：　B

　　6、存货生产是按照（ ）划分的。

　　A、生产技术特点

　　B、接受生产任务方式

　　C、生产连续程度

　　D、生产的重复程度和工作地专业化程度

　　参考答案：　B

　　7、间断生产方式是按照（ ）划分的

　　A、生产技术特点

　　B、接受生产任务方式

　　C、生产连续程度

　　D、生产的重复程度和工作地专业化程度

　　参考答案：　C

　　8、2、连续生产是指（ ）。

　　A、生产各要素是间断投入的。

　　B、产品的工艺流程、使用的设备是固定的，按流程顺序连续进行的。

　　C、在制品有储备

　　D、一天开两班

　　参考答案：　B

　　9、采用专用设备和专用工艺装备是（ ）特点

　　A、大量生产

　　B、单件生产

　　C、成批生产

　　D、单件小批生产

　　参考答案：A

　　10、采用通用设备和通用的工艺装备是（ ）特点。

　　A、大量大批生产

　　B、单件小批生产

　　C、成批生产

　　D、流水生产

　　参考答案：　B

　　11、单件生产是按照（ ）划分的。

　　A、生产技术特点

　　B、接受生产任务方式

　　C、生产连续程度

　　D、生产的重复程度和工作地专业化程度

　　参考答案：　D

　　1、基本生产单位如（ ）。

　　A、分厂、车间、工段、班组

　　B、总公司、职能部门

　　C、职能部门、车间、运输部门

　　D、销售部门、保卫部门

　　参考答案：A

　　2、按工艺专业化形式设置的生产单位要完成产品的整个加工过程，（ ）。

　　A、需要跨生产车间

　　B、不需要跨生产车间

　　C、大部分产品不需要跨生产车间

　　D、大部分产品需要跨生产车间

　　参考答案：　D

　　3、对产品品种的变换有强的适应性是（ ）的优点。

　　A、工艺专业化

　　B、对象专业化

　　C、流水线

　　D、单一品种流水线

　　参考答案：A

　　4、对产品品种的变换有较强的适应性是（ ）的优点。

　　A、对象专业化

　　B、流水生产

　　C、混流生产

　　D、无人工厂

　　参考答案：　C

　　5、生产过程要满足连续性、平行性、比例性、单向性、均衡性、适应性是（ ）。

　　A、组织生产过程的要求

　　B、生产计划的要求

　　C、组织设计流水线的要求

　　D、设备管理的要求

　　参考答案：A

　　6、自然过程是生产过程的一个组成部分，如（ ）。

　　A、油漆的干燥

　　B、人寻找工具

　　C、钢铁的冶炼

　　D、纺纱织布

　　参考答案：A

　　7、按产品、部件、零件来设置生产单位，是（ ）的特点。

　　A、工艺专业化

　　B、对象专业化

　　C、开放车间

　　D、装配车间

　　参考答案：　B

　　8、（ ）也称封闭车间

　　A、按工艺专业化来设置的生产车间

　　B、按生产过程的先后顺序来设置的车间

　　C、按对象专业化来设置的生产车间

　　D、按反工艺顺序来设置的生产车间

　　参考答案：　C

　　9、缩短产品加工路线；减少仓库和生产面积占用；提高生产过程的连续性；减少车间的生产联系是（ ）优点。

　　A、工艺专业化

　　B、对象专业化

　　C、单一品种生产

　　D、多品种生产

　　参考答案：　B

　　10、对市场反应迟钝，是（ ）的缺点

　　A、柔性制造

　　B、混流生产

　　C、对象专业化

　　D、工艺专业化

　　参考答案：　C

　　11、企业生产的专业方向比较稳定；生产规模大；设备工种齐全配套；原材料供应稳定；市场需求量多，适合采用（ ）。

　　A、工艺专业化形式

　　B、对象专业化形式

　　C、订货生产

　　D、间断生产

　　参考答案：　B

　　12、企业生产的专业方向不稳定；市场变化莫测；原材料供应不稳，生产规模较小，宜采用（ ）。

　　A、定货生产

　　B、对象专业化形式

　　C、连续生产

　　D、流水生产

　　参考答案：A

　　1、使加工对象以一定的速度（节拍）连续不断的通过各个工作地进行加工，并出产产品的一种生产组织形式叫（ ）。

　　A、对象专业化

　　B、工艺专业化

　　C、流水生产

　　D、成组技术

　　参考答案：　C

　　2、流水线的特点有（ ）。

　　A、工作地专业化程度很高

　　B、工作地专业化程度很低

　　C、加工顺序按反工艺顺序排列

　　D、加工顺序无一定规律

　　参考答案：A

　　3、工作地的专业化程度高；劳动对象按加工的工艺顺序排列；按照统一规定的节拍出产产品是（ ）特点。

　　A、流水线的特点

　　B、订货生产

　　C、存货生产

　　D、连续生产

　　参考答案：A

　　4、按流水线上生产对象的移动方式划分有（ ）。

　　A、产品固定不移动流水线，产品移动流水线

　　B、单一品种流水线，多品种流水线

　　C、连续流水线，间断流水线

　　D、强制节拍流水线，自由节拍流水线

　　参考答案：A

　　5、不变流水线，可变流水线是按（ ）划分。

　　A、流水线上生产对象的移动方式

　　B、流水线的生产对象数目

　　C、流水线上生产对象的轮换方式

　　D、流水线上生产过程的连续程度。

　　参考答案：　C

　　6、流水线的优点有（ ）。

　　A、生产效率高，经济效益好

　　B、适应市场变化快

　　C、改进流水线花费较少

　　D、工人不易疲劳

　　参考答案：A

　　7、流水线的缺点有（ ）。

　　A、生产效率低

　　B、建设和改进流水线费用高

　　C、管理工作复杂

　　D、经济效益差

　　参考答案：　B

　　8、产品品种稳定；产品结构先进；原材料外购件是标准化的；适宜采用（ ）。

　　A、成批生产

　　B、单件生产

　　C、混流生产

　　D、流水生产

　　参考答案：　D

　　9、机器设备经常处于完好状态；工作质量稳定；社会需求稳定且量大；原材料，外购件易得到，适宜采用（ ）。

　　A、订货生产

　　B、单件生产

　　C、柔性制造

　　D、流水生产

　　参考答案：　D

　　（答案仅供参考）
