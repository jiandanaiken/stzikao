　　1、在工资等级的基础上，根据每个职工的劳动耗费或生产成果，计算劳动量并致富工资的具体形式，称为（ ）。

　　A 工资制度

　　B 工资水平

　　C 工资形式

　　D 工资管理

　　答案　C

　　2、根据职工工资等级所示的工资额，按实际劳动时间来计算并支付报酬的工资形式是（ ）。

　　A 计件工资

　　B 计时工资

　　C 浮动工资

　　D 结构工资

　　答案　B

　　3、计时工资决定职工工资额多少的因素是工资等级和（ ）。

　　A 劳动时间

　　B 工种

　　C 劳动态度

　　D 劳动场所

　　答案　A

　　4、计时工资以劳动时间来计算报酬，前提条件是（ ）。

　　A 职工工资等级标准合理

　　B 职工自愿接受等级标准

　　C 全体职工一个标准

　　D 工作时间可长可短

　　答案　A

　　5、计时工资的缺陷表现在（ ）。

　　A 不能表现职工工作性质

　　B 不能反映职工实际作业能力和熟练程度

　　C.不能浮动

　　D 不能激励职工多干

　　答案　B

　　6、根据职工完成合格产品的数量和作业数量，按既定的计件单价计算支付职工报酬的方式是（ ）。

　　A 浮动工资

　　B 计时工资

　　C 奖金

　　D 计件工资

　　答案　D

　　7、企业中较为适合采用计件工资的条件是（ ）。

　　A 自动化高的工种

　　B 独立性的工种

　　C 流水作业的工种

　　D 手工操作为主的工种

　　答案　D

　　8、计件工资是计时工资的转化形式，它能反映（ ）。

　　A 不同等级工人间的劳动差别

　　B 不同时间的劳动差别

　　C 不同等级间和同等级工人劳动的差别

　　D 超额劳动

　　答案　C

　　9、不论工人完成和超额完成定额的程度如何，均按同一单价计算工资的形式是（ ）。

　　A 直接无限计件工资制

　　B 直接有限计件工资制

　　C 累进计件工资制

　　D 超定额计件工资制

　　答案　A

　　10、对计件超额工资规定一个最高限额的计件工资称为（ ）。

　　A 直接无限计件工资制

　　B 直接有限计件工资制

　　C 累进计件工资制

　　D 超定额计件工资制

　　答案　B

　　11、把计时工资和计件工资结合起来运用的形式是（ ）。

　　A 直接无限计件工资制

　　B 直接有限计件工资制

　　C 累进计件工资制

　　D 超定额计件工资制

　　答案　D

　　（答案仅供参考）
