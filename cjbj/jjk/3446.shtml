　　1、企业"战略制胜"起始于（ ）

　　A 四十年代

　　B 五十年代

　　C 六十年代

　　D 七十年代

　　参考答案：　D

　　2、调整产品结构，改变单一品种的格局，向产品多样化发展。企业这一战略方向的发展适应了（ ）

　　A 需求结构变化的要求

　　B 科技进步的要求

　　C 资源供应紧张的要求

　　D 社会政治形势多变的要求

　　参考答案：A

　　3、企业面对激烈变化的环境，严峻挑战的竞争，为谋求生存和不断发展而作出的总体性、长远性的谋划和方略。这一概念是指 （ ）

　　A 经营战略

　　B 经营哲学

　　C 经营宗旨

　　D 经营方针

　　参考答案：A

　　4、经营战略的实质在于寻求（ ）

　　A 产品的革新

　　B 技术的革新

　　C 营销的革新

　　D 企业的革新

　　参考答案：　D

　　5、经营战略是以企业整体为对象，是根据企业总体的发展需要而制定的。这一特征体现了经营战略的 （ ）

　　A 长远性

　　B 全局性

　　C 纲领性

　　D 抗争性

　　参考答案：　B

　　6、经营战略是对企业未来一定时期生存和发展的统筹谋划，它着眼于企业的长远发展，追求的是企业长期利益。这一特征体现了经营战略的 （ ）

　　A 长远性

　　B 全局性

　　C 纲领性

　　D 抗争性

　　参考答案：A

　　7、居于企业管理周期的中心环节的是 （ ）

　　A 战略制定的准备工作

　　B 战略方案的拟定、评价和选择工作

　　C 战略的实施工作

　　D 战略的控制工作

　　参考答案：　B

　　8、在企业管理中处于核心地位的是（ ）

　　A 生产制造管理工作

　　B 产品营销管理工作

　　C 技术开发管理工作

　　D 经营战略决策工作

　　参考答案：　D

　　（答案仅供参考）
