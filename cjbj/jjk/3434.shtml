  • 09年自考“企业管理概论”综合题（4）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、管理产生于人类的（ ）。

　　A 个体劳动

　　B 社会组织

　　C 共同劳动

　　D 自然发展

　　答案　C

　　2、管理活动包括（ ）。

　　A 投资

　　B 经营

　　C 销售

　　D 计划、组织、协调

　　答案　D

　　3、管理是（ ）的必然产物。

　　A 社会化劳动，分工协作

　　B 私有制

　　C 商品经济

　　D 个人劳动

　　答案　A

　　4、企业经营是以企业取得理想的（ ）为目的的。

　　A 社会效益

　　B 经济效益

　　C 价值

　　D 发展规模

　　答案　B

　　5、企业的一切活动是以（ ）为中心的。

　　A 生产

　　B 管理

　　C 经营

　　D 销售

　　答案　C

　　6、管理作为一种独立的社会职能是从（ ）中逐步独立出来的。

　　A 指挥

　　B 经营

　　C 监督

　　D 计划

　　答案　B

　　7、管理对经营有（ ）作用。

　　A 指挥

　　B 滞后

　　C 监督

　　D 驾驭

　　答案　D

　　8、马克思主义认为，企业管理具有（ ）。

　　A 价值和使用价值

　　B 完全同一性

　　C 自然属性和社会属性

　　D 阶段性和短期性

　　答案　C

　　9、管理与生产力、社会化大生产相联系的属性是（ ）。

　　A 组织技术属性

　　B 社会属性

　　C 社会经济属性

　　D 阶级属性

　　答案　A

　　10、社会经济属性是指管理与（ ）相联系。

　　A 生产利

　　B 生产关系、企业制度

　　C 企业制度

　　D 社会化大生产

　　答案　B

　　11、马克思主义认为，资本主义生产是生产（ ）的过程。

　　A 资本主义生产关系

　　B 生产力

　　C 物质产品

　　D.物质产品和资本主义生产关系

　　答案　D

　　12、管理代表着生产资料所有者的利益和目的，这反映出管理的（ ）

　　A 自然属性

　　B 社会属性

　　C 根本利益

　　D 二重性

　　答案　B

　　13、马克思主义认为，企业管理具有（ ）

　　A 二重性

　　B 时效性

　　C 多元化

　　D 管理统一性

　　答案　A

　　14、各个不同的生产方式中，（ ）的属性是基本相同的。

　　A 社会

　　B 组织生产力

　　C 社会经济

　　D 管理

　　答案　B

　　15、对于资本主义发达国家的先进经营方式和管理方法，我们的态度应 是（ ）。

　　A 被动吸收

　　B 主动批判

　　C 全面吸收

　　D 学习、批判、继承、创新

　　答案　D

　　16、西方资本主义的管理理论和经验，既有科学合理的一面，又蕴藏 着虚伪、残酷的（ ）本质。

　　A 剥削

　　B 创造

　　C 欺诈

　　D 增值

　　答案　A

　　17、企业管理客观上存在自然属性和社会属性，所以在管理实践中，必 须（ ）。

　　A 实现生产要素的优化组合

　　B 不断巩固社会主义生产关系

　　C 解决企业所有者、管理者、劳动者的相互关系

　　D 以上三点

　　答案　D

　　18、社会主义企业管理的任务是（ ）。

　　A 重视生产力的发展

　　B 重视生产力和与之相适应的生产关系的联系

　　C 重视生产关系的发展

　　D 自力更生、独立自主

　　答案　B

　　19、运用马克思主义管理二重性原理，在管理实践中，要（ ）。

　　A 重视生产关系的发展

　　B.自力更生、独立自主

　　C 重视生产力和与之相适应的生产关系的联系

　　D 重视生产力的发展

　　答案　C

　　20、管理的思想和方法是（ ）

　　A 一成不变的

　　B 完全否定过去的

　　C 不断发展和创新的

　　D 颂古非今的

　　答案　C

　　21、建立起具有中国特色的社会主义企业管理科学体系，必须充分重视（ ）

　　A 科学管理

　　B 发挥公有制的优越性

　　C 发挥优良传统，学习和借鉴

　　D 以上三点

　　答案　D

　　22、随着社会生产力和生产关系的发展和变化，（ ）必须不断做出调整。

　　A 以下三点

　　B 管理思想和观念

　　C 管理方式和手段

　　D 管理理论和机制

　　答案　A

　　23、企业管理职能是对企业管理的基本（ ）所做的理论概括。

　　A 工作内容和工作过程

　　B 工作业绩和工作水平

　　C 工作数量和工作质量

　　D 工作方法和工作理念

　　答案　A

　　24、企业管理具有（ ）。

　　A 特殊职能

　　B 一般职能

　　C 一般职能和特殊职能

　　D 以上全否

　　答案　C

　　25、企业管理的（ ）在企业中有着共性的规律。

　　A 特殊职能

　　B 一般职能

　　C 一般职能和特殊职能

　　D 以上全否

　　答案　B

　　26、企业管理调整所有者、经营管理者和劳动者之间的生产关系和利 益，这是企业的（ ）。

　　A 特殊职能

　　B 一般职能

　　C 一般职能和特殊职能

　　D 以上全否

　　答案　A

　　27、对企业未来的管理活动进行筹划和安排是管理的（ ）职能。

　　A 激励

　　B 计划

　　C 控制

　　D.指挥

　　答案　B

　　28、企业管理活动的首要职能是（ ）。

　　A 控制职能

　　B 指挥职能

　　C 激励职能

　　D 计划职能

　　答案　D

　　29、管理计划职能的主要内容是（ ）。

　　A 调查研究、预测分析

　　B 战略决策

　　C 计划实施

　　D 包括以上的"四位一体"

　　答案　D

　　30、对实现企业目标的各种要素和人们在经济活动中相互关系进行组 合、配置的活动是企业管理的（ ）职能。

　　A 组织

　　B 计划

　　C 控制

　　D.指挥

　　答案　A

　　31、组织职能的内容一般包括（ ）。

　　A 确定各项具体管理职能 、确定协调方式和控制手段

　　B 确定各个管理层次、部门、岗位及其权益

　　C 配备和训练管理人员，制定各项规章制度

　　D.以上三点

　　答案　D

　　32、为组织职能规定了方向乃至具体要求的是（ ）。

　　A 组织

　　B 计划

　　C 控制

　　D 指挥

　　答案　B

　　33、人员的选拔、使用、考核、奖惩和培养等一系列管理活动是（ ）。

　　A 控制职能

　　B 指挥职能

　　C 激励职能

　　D 计划职能

　　答案　C

　　34、企业最重要的资源是（ ）资源。

　　A 物质

　　B 社会

　　C 自然

　　D 人力
