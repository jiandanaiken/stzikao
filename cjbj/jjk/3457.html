  • 09年自考“企业管理概论”练习题（12）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、不同专业化形式的车间，计划编制方法是不同的。如果是（ ）车间，每个车间分别独立地完成一定产品的全部（或接近全部）生产过程，编制车间作业计划的方法，基本上按照各个车间既定的专业分工来分配生产任务。

　　A 对象专业化

　　B 工艺专业化

　　C 产品生产加工

　　D 装配

　　参考答案：A

　　2、如果是（ ）车间，各车间里依次加工半成品的关系。分配各车间的生产任务，既要能够保证企业的商品生产任务按期、按量地完成，又要确保各个车间之间在生产期限和生产数量上的衔接协调。

　　A 对象专业化

　　B 工艺专业化

　　C 产品生产加工

　　D 装配

　　参考答案：　B

　　3、在大量大批生产企业，产品品种单一，产量比较大，工艺和各车间的协作关系比较稳定，可采用（ ）方法编制生产作业计划

　　A 累计编号法

　　B 在制品定额法

　　C 生产周期法

　　D 提前期法。

　　参考答案：　B

　　4、在制品定额法编制车间作业计划经常用到的期量有（ ）。

　　A 库存半成品定额、批量

　　B 库存半成品定额、生产周期

　　C 在制品定额、库存半成品定额

　　D 批量、提前期

　　参考答案：　C

　　5、（ ）适用于成批轮番生产企业。在这类企业中，由于产品轮番上下场，因而各个生产环节结存的在制品的品种和数量经常有变化，不能采用在制品定额法。

　　A 提前期法

　　B 生产周期化

　　C MRP法

　　D 在制品定额法

　　参考答案：A

　　6、（ ）它是指产品（零件）在各车间出产（或投入）的时间同成品生产时间相比，所要提前的天数。

　　A 提前期

　　B 生产周期

　　C 在制品定额

　　D 库存半成品定额。

　　参考答案：A

　　7、产品的（ ），是指产品从原材料投入生产起，一直到成品生产为止的全部日历时间。

　　A 批量

　　B 在制品定额

　　C 生产周期

　　D 提前期

　　参考答案：　C

　　8、生产批量是（ ）。

　　A 相同产品一次投入和生产的数量

　　B 相同产品一次投入的数量

　　C 相同产品一次出产的数量

　　D 相同产品出产和投入的数量。

　　参考答案：A

　　9、生产技术文件准备；原材料和外协件的准备；机电设备的检修准备属于（ ）内容。

　　A 生产作业计划

　　B 生产准备工作

　　C 生产调度

　　D 生产计划

　　参考答案：　B

　　10、工艺装备的设计，制造和检修；人员方面的准备属于（ ）内容。

　　A 生产作业计划

　　B 生产准备工作

　　C 生产调度

　　D 生产计划

　　参考答案：　B

　　11、（ ）根据工段和班组的生产作业计划以及实际生产进度，为各个工作地和工人具体地分派生产任务，并做好作业准备工作，以促使生产任务顺利开展的活动。

　　A 生产计划

　　B 日常生产派工

　　C 生产作业计划

　　D 生产调度

　　参考答案：　B

　　12、（ ）又称工票，或作业传票。是最基本的生产凭证之一。

　　A 单工序工票

　　B 加工路线单

　　C 派工单

　　D 看板

　　参考答案：　C

　　13、（ ）是把企业的全年生产任务，具体地分配到各车间、工段、班组以至每个工作地和个人，规定他们在每月、每旬、每周、每日以至每个轮班和小时内的具体生产任务，从而保证按品种、质量、数量、期限和成本完成企业的生产任务。

　　A 生产作业计划

　　B 生产计划

　　C 车间作业计划

　　D 生产调度

　　参考答案：A

　　14、编制生产作业计划的工作内容和作用、车间作业计划的编制、MRP法、网络计划技术、生产作业控制是（ ）章的内容

　　A 全面质量管理

　　B 设备管理

　　C 生产作业计划与控制

　　D 生产计划

　　参考答案：　C

　　15、编制全厂的生产作业计划和车间内部的作业计划；编制生产准备计划；进行设备的负荷核算和平衡；日常派工；制定和修改期量标准是（ ）工作内容。

　　A 劳动计划

　　B 生产调度

　　C 编制车间作业计划

　　D 编制生产作业计划

　　参考答案：　D

　　16、（ ）又叫作业计划标准，它是为制造对象在生产期限和生产数量方面所规定的一种标准数据。

　　A 期量标准

　　B 提前期

　　C 生产周期

　　D 批量

　　参考答案：A

　　17、MRP的目标是（ ）。

　　A 保证按时供应用户所需产品。

　　B 保证按时供应用户所需产品，及时取得生产所需原材料及零部件；

　　C 保证尽可能低的库存水平

　　D 保证按时供应用户有所需产品，及时取得生产所需原材料及零部件；保证尽可能低的库存水平；计划生产活动、交货进度与采购活动，使各车间生产的零部件，外购配套件与装配的要求在时间和数量上精确衔接。

　　参考答案：　D

　　18、MRP的组成有（ ）。

　　A 主生产进度计划

　　B 物料清单 C 主生产进度计划；

　　C物料清单；存储记录；MRP输出；能力的计划与控制

　　D 能力
