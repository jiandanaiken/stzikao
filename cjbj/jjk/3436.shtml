  • 09年自考“企业管理概论”综合题（6）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、若干相互作用和相互依赖的组成部分综合而成的，具有特定功能的有机体称为（ ）。

　　A 组织

　　B 团体

　　C 系统

　　D产品线

　　答案　C

　　2、系统论的思想在企业管理中的应用是一种（ ）。

　　A 措施

　　B 具体方法

　　C 创意

　　D 科学理论与方法

　　答案　D

　　3、企业管理作为一个系统，具有集合性、相关性、目的性和（ ）等特点。

　　A 分离性

　　B 环境适应性

　　C 模糊性

　　D不可控性

　　答案　B

　　4、企业管理作为一个系统，具有集合性、相关性、环境适应性和（ ）等特点。

　　A 目的性

　　B 独立性

　　C 模糊性

　　D不可控性

　　答案　A

　　5、经过加工处理后，对企业生产经营活动有价值的数据和资料称为（ ）。

　　A 企业档案

　　B 企业管理信息

　　C 企业报告

　　D企业通知

　　答案　B

　　6、企业管理的信息源来自（ ）。

　　A 外部环境和企业内部

　　B 企业上级和基层

　　C 政府和社会

　　D 职工和顾客

　　答案　A

　　7、企业信息工作的基本要求是完整性、准确性和（ ）

　　A 充足性、历史性

　　B 局部性、阶段性

　　C 及时性、适应性

　　D 阶段性、重点性

　　答案　C

　　8、企业管理的控制过程包括拟定控制标准、衡量实际成果和（ ）等三个基本阶段。

　　A 建立控制体系

　　B 培训控制人员

　　C 估算控制费用

　　D采取措施

　　答案　D

　　9、企业管理的控制过程包括采取措施、衡量实际成果和（ ）等三个基本阶段。

　　A 建立控制体系

　　B 拟定控制标准

　　C 估算控制费用

　　D培训控制人员

　　答案　B

　　10、企业管理的控制过程包括拟定控制标准、采取措施和（ ）等三个基本阶段。

　　A 衡量实际成果

　　B 建立控制体系

　　C 估算控制费用

　　D培训控制人员

　　答案　A

　　11、企业管理中，揭示目标实施过程中的偏差，实现对系统运行的控制是通过（ ）实现的。

　　A 领导指示

　　B 信息反馈

　　C 组织机构

　　D人员调动

　　答案　B

　　12、企业管理的控制方式根据信息反馈的方式划分，可以分为（ ）两种类型。

　　A 人员控制和自动控制

　　B 组织控制和个人控制

　　C 前馈控制和反馈控制

　　D 计划控制和非计划控制

　　答案　C

　　13、运用行政权力，通过指令和隶属关系确立行为规范的管理方法是（ ）。

　　A 经济方法

　　B 法律方法

　　C 教育方法

　　D 行政方法

　　答案　D

　　14、行政方法的实质是（ ）。

　　A 上级对下级的干预，下级对上级的服从

　　B 物质利益的诱导

　　C 权威的直接规范

　　D 改变人的世界观

　　答案　A

　　15、行政方法的特点有强制性、直接性、具体性和（ ）。

　　A 稳定性

　　B 间接性

　　C 弥散性

　　D 机动灵活性

　　答案　D

　　16、经济方法的实质是（ ）。

　　A 上级对下级的干预，下级对上级的服从

　　B 物质利益的诱导

　　C 权威的直接规范

　　D 改变人的世界观

　　答案　B

　　17、利用经济杠杆、通过物质利益的调节，实现企业目标方法是（ ）。

　　A 经济方法

　　B 法律方法

　　C 思想教育方法

　　D 行政方法

　　答案　A

　　18、经济方法的特点是诱导性、渐进性（ ）。

　　A 稳定性

　　B 间接性

　　C 弥散性

　　D 机动灵活性

　　答案　B

　　19、法律方法的实质是（ ）。

　　A 上级对下级的干预，下级对上级的服从

　　B 物质利益的诱导

　　C 权威的直接规范

　　D 改变人的世界观

　　答案　C

　　20、利用立法和执法，通过法律程序调节进关系，达到管理目标的方法（ ）。

　　A 经济方法

　　B 法律方法

　　C 思想教育方法

　　D 行政方法

　　答案　B

　　21、法律方法的特点是强制性、权威性、明确性和（ ）。

　　A 稳定性

　　B 间接性

　　C 弥散性

　　D 机动灵活性

　　答案　A

　　（答案仅供参考）
