　　1、社会主义工资分配制度改革的原则是物质利益原则和（ ）

　　A 按劳分配原则

　　B 按需分配原则

　　C 平均主义原则

　　D 精神激励原则

　　答案　A

　　2、社会主义工资分配制度改革的原则是按劳分配原则和（ ）

　　A 按需分配原则

　　B 物质利益原则

　　C 平均主义原则

　　D 精神激励原则

　　答案　B

　　3、社会主义工资分配制度通过物质利益原则，要调动（ ）的积极性。

　　A 国家

　　B 企业

　　C 国家、企业、职工

　　D 职工

　　答案　C

　　4、我国等级工资制分为工人等级工资制度和（ ）。

　　A 领导等级工资制度

　　B 工种等级工资制度

　　C 职业等级工资制度

　　D 职务等级工资制度

　　答案　D

　　5、职务等级工资制度是用于（ ）

　　A 企业工人

　　B 企业班组

　　C 国家机关工作人员、企业管理人员工程技术人员

　　D 部队

　　答案　C

　　6、工人等级工资根据工人从事的工作的复杂程度、劳动繁重程度、工 作责任大小和（ ）。

　　A 工作环境

　　B 工人技术熟练程度

　　C 工作年限

　　D 工作内容

　　答案　B

　　7、我国等级工资制度曾经起过积极作用，但也存在一定弊端，归纳起来是（ ）。

　　A "低、平、死、乱"

　　B "少、乱、平、差"

　　C "高、乱、平、活"

　　D "四、平、高、差"

　　答案　A

　　8、我国等级工资制度存在的问题是"低、平、死、乱"，其中"死"是指（ ）。

　　A 管理方法

　　B 管理体制

　　C 管理手段

　　D 管理标准

　　答案　B

　　9、我国等级工资制度存在的问题是"低、平、死、乱"，其中"低"是 （ ）。

　　A 管理水平

　　B 职工要求

　　C 职工素质

　　D 工资水平

　　答案　D

　　10、在社会主义市场经济条件下，在分配制度方面，应坚持（ ）的原则。

　　A 多种分配为主体

　　B 按劳分配为主体，多种分配方式并存

　　C 按资分配为主体

　　D 按资分配为主体，按劳分配为辅

　　答案　B

　　11、坚持按劳分配的原则是为了（ ）激励职工努力提高业务水平。（ ）。

　　A 鼓励先进，照顾落后

　　B 统一标准，平均分配

　　C 鼓励先进，鞭策落后

　　D 稳定先进

　　答案　C

　　12、我国工资制度体系中，主要分配形式是（ ）。

　　A 奖金

　　B 津贴

　　C 福利

　　D 工资

　　答案　D

　　13、我国工资制度的改革方向是建立以工资为主体的，内在科学完备的工资制度，（ ）。

　　A 全面体现主体形态的劳动

　　B 全面体现不同形态的劳动

　　C 体现某一形态的劳动

　　D 部分体现不同形态的劳动

　　答案　B

　　14、作为对工资的必要报酬，超额劳动的报酬是（ ）。

　　A 津贴

　　B 福利

　　C 奖金

　　D 计件工资

　　答案　C

　　15、工资制度建设是提高企业管理水平的重要途径，它可以发挥激励作用、凝聚作用和（ ）。

　　A 统计作用

　　B 树立管理权威的作用

　　C 约束作用

　　D 分权作用

　　答案　B

　　16、工资制度建设是提高企业管理水平的重要途径，它可以发挥激励 作用、树立管理权威作用和（ ）。

　　A 约束作用

　　B 分权作用

　　C 统计作用

　　D 凝聚作用

　　答案　D

　　17、工资制度建设是提高企业管理水平的重要途径，它可以发挥凝聚的作用、树立管理权威的作用和（ ）。

　　A 激励作用

　　B 分权作用

　　C 统计作用

　　D 约束作用

　　答案　A

　　18、市场经济对企业工资制度的基本要求是劳动报酬要（ ）。

　　A 货币化、工资化

　　B 统一化、平均化

　　C 复杂化、规范化

　　D 简单化、多样化

　　答案　A

　　19、我国工资制度改革的方向是（ ）。

　　A "小工资、大奖金"

　　B 使工资居职工收入的主导地位

　　C 逐步取消奖金

　　D 不断扩大福利的比重

　　答案　B

　　20、我国工资制度改革的方向是实现（ ）的良性循环的模式。

　　A 低工资 - 高效率

　　B 高工资 - 低效率

　　C 低工资 - 低效率 - 高工资 - 高效率

　　D 高效率 - 高工资 - 更高效率 - 更高工资

　　答案　D

　　21、我国工资制度发展方向是（ ）相联系。

　　A 高工资与低就业

　　B 高工资与高效率

　　C 低效率与高工资

　　D 低工资与高效率

　　答案　B

　　（答案仅供参考）
