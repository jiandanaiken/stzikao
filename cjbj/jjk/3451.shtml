  • 09年自考“企业管理概论”练习题（6）
  • 发布日期时间：2009-4-6  来源：zikao365   点击数：  作者：本站整理

　　1、流水生产组织、生产类型、基本生产单位的组织形式是（ ）章的内容。

　　A、生产过程组织

　　B、劳动组织

　　C、质量管理

　　D、设备管理

　　参考答案：A

　　2、生产过程组织的要求、生产类型是（ ）章的内容。

　　A、劳动定额

　　B、生产过程组织

　　C、全面质量管理

　　D、设备管理

　　参考答案：　B

　　3、生产类型是（ ）章的内容。

　　A、全面质量管理

　　B、生产计划

　　C、生产作业计划与控制

　　D、生产过程组织

　　参考答案：　D

　　4、最终产品是经过生产和加工后，（ ）。

　　A、再参加生产过程，满足生产和生活的需要

　　B、不再参加生产过程，满足生产和生活的需要

　　C、再参加生产过程，满足生产的需要

　　D、不再参加生产过程，满足生活的需要

　　参考答案：　B

　　5、生产技术准备过程是指产品投入生产前的各项生产技术准备工作。如（ ）。

　　A、设备维修

　　B、产品设计、工艺设计、劳动定额的制定

　　C、成品的供应、保管

　　D、炼钢、轧钢

　　参考答案：　B

　　6、辅助生产过程是指为保证基本生产过程所进行的各种辅助性产品的生产过程，以及辅助性的生产活动，如：（ ）

　　A、设备维修，工具制造。

　　B、产品设计、工艺设计、劳动定额的制定。

　　C、成品的供应、保管。

　　D、炼钢、轧钢。

　　参考答案：A

　　7、可以缩短生产周期，减少在制品的占用量，可以更好的利用物资，设备和生产面积，减少由于产品停顿、等待所造成的损失，有利于改善产品的质量，是保持和提高生产过程的（ ）的好处。

　　A、连续性

　　B、比例性

　　C、适应性

　　D、均衡性

　　参考答案：A

　　8、生产过程的连续性，同（ ）有关。

　　A、财务状况

　　B、技术水平、管理水平

　　C、土地状况

　　D、信息资源

　　参考答案：　B

　　9、平行性是（ ）的前提条件。

　　A、单向性

　　B、适应性

　　C、连续性

　　D、比例性

　　参考答案：　C

　　10、平行性有利于（ ）

　　A、减少运输

　　B、均衡性

　　C、单向性

　　D、节省时间

　　参考答案：　D

　　11、基本生产过程，按工艺加工的性质，可分为三个基本阶段，即（ ）。

　　A、结束阶段、准备阶段、加工阶段

　　B、准备阶段、结束阶段、装配阶段

　　C、准备阶段、加工阶段、装配阶段

　　D、准备阶段、装配阶段、结束阶段

　　参考答案：　C

　　12、工序按其作用可分为（ ）

　　A、工艺工序、检验工序、运输工序

　　B、加工工序、保险工序、运输工序

　　C、工艺工序、保险工序、运输工序

　　D、检验工序、保险工序、运输工序

　　参考答案：A

　　13、生产过程的单向性有利于（ ）。

　　A、减少搬运量

　　B、平行性

　　C、均衡性

　　D、减少加工时间

　　参考答案：A

　　14、生产过程的单向性有利于（ ）。

　　A、平行性

　　B、比例性

　　C、减少运输时间

　　D、减少加工时间

　　参考答案：　C

　　15、比例性就是要求（ ）。

　　A、生产过程各能力要协调，平衡，不要有的能力大，有的能力小，差距悬殊。

　　B、生产过程各工序运输要有节奏

　　C、生产过程各项资源配置均等

　　D、生产过程各项资源可以有高有低

　　参考答案：A

　　16、为了保持生产过程的比例性，可以（ ）。

　　A、使各项能力完全保持一致。

　　B、使各生产环节保持必要的比例关系，若生产能力发生变化，可以通过协作、特许、有偿加工等方法解决。

　　C、用增大原材料、动力设备投入的办法来解决。

　　D、用增加产品的产出办法来解决。

　　参考答案：　B

　　17、充分利用设备和人力资源，避免加班加点，实现安全生产和文明生产，这是（ ）的好处。

　　A、连续性

　　B、适应性

　　C、均衡性

　　D、平行性

　　参考答案：　C

　　18、均衡生产是指（ ）。

　　A、适应市场变化

　　B、生产进度要保持均衡有节奏

　　C、各生产环节有比例

　　D、各生产环节要连续

　　参考答案：　B

　　19、生产过程的适应性是指（ ）。

　　A、生产过程各环节要连续

　　B、灵活应变，适应市场要求的变化

　　C、生产过程总是加工固定不变的产品

　　D、生产过程各环节保持比例关系不变化

　　参考答案：　B

　　20、（ ）是生产过程适应性的体现。

　　A、刚性生产

　　B、无人工厂

　　C、柔性生产

　　D、流水生产

　　参考答案：　C

　　21、以最低的总成本，为可靠地实现产品或作业的必要功能所进行的、着重于功能分析的有组织的活动。这是指（ ）

　　A 网络计划技术

　　B 成本分析

　　C 价值工程

　　D 技术开发

　　参考答案：　C

　　22、通过对产品功能的分析，正确处理功能与成本关系来降低成本（费用）的科学方法。这是指（ ）

　　A 网络计划技术

　　B 成本分析方法

　　C 价值工程

　　D 技术开发方法

　　参考答案：　C

　　23、在价值工程中，提高产品的价值的途径有（ ）

　　A 功能不变，使成本降低

　　B 增加成本，使功能提高

　　C 功能下降，成本下降

　　D 成本不变，功能不变

　　参考答案：A

　　24、在价值工程中，产品的价值是指（ ）

　　A 使用价值

　　B 功能与成本之比

　　C 技术先进性

　　D 寿命周期成本

　　参考答案：　B

　　25、产品的功能就是指产品的（ ）

　　A 使用价值

　　B 价值

　　C 技术

　　D 成本

　　参考答案：A

　　26、价值工程中所讲的产品功能是指产品的（ ）

　　A 使用功能

　　B 必要功能

　　C 基本功能

　　D 辅助功能

　　参考答案：　B

　　27、价值工程中所讲的寿命周期成本是指产品的（ ）

　　A 制造成本

　　B 使用成本

　　C 设计成本

　　D 制造成本和使用成本

　　参考答案：　D

　　28、价值工程中所讲的产品寿命周期是指产品（ ）

　　A 使用寿命

　　B 市场寿命

　　C 从研制、生产、销售、使用直到报废的整个时期

　　D

　　参考答案：　C

　　29、价值工程的目的在于（ ）

　　A 提高功能、成本之比

　　B 进行功能分析

　　C 降低产品成本

　　D 提高产品功能

　　参考答案：A

　　30、开展价值工程要以（ ）

　　A 功能分析为核心

　　B 降低产品成本为核心

　　C 提高产品功能为核心

　　D 技术开发为核心

　　参考答案：A

　　（答案仅供参考）
