  • 09年自考“企业管理概论”综合题（13）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、我国工资制度改革以来比较普遍使用的，代表着工资改革方向的是（ ）。

　　A 计件工资

　　B 计时工资

　　C 浮动工资

　　D 结构工资

　　答案　D

　　2、结构工资有成组合工资制，包括：基础工资、职务（岗位）工资、年功工资和（ ）。

　　A 计件工资

　　B 奖励工资

　　C 加班工资

　　D 计时工资

　　答案　B

　　3、结构工资有成组合工资制，包括：基础工资、职务（岗位）工资、 奖励工资和（ ）。

　　A 计件工资

　　B 计时工资

　　C 加班工资

　　D 年功工资

　　答案　D

　　4、工资作为劳动报酬，必须取决于（ ）。

　　A 职工劳动形态

　　B 职工劳动状态

　　C 职工劳动方式

　　D 职工素质

　　答案　A

　　5、职工劳动形态可以划分为（ ）等三种形态。

　　A 长期的、短期的、永久的

　　B 潜在的、流动的、凝固的

　　C 单独的、合伙的、规模的

　　D 私有的、公有的、共有的

　　答案　B

　　6、我国工资制度改革应以结构工资为基本原理，以（ ）为原则 来构造工资制度。

　　A 岗效固定

　　B 平均分配

　　C 拉开档次

　　D 岗效挂钩浮动

　　答案　D

　　7、将职工基本工资的一部分或全部与奖金等结合起来，随企业经济效益和职工劳动成果变化上下浮动的工资制度是（ ）。

　　A 计件工资

　　B 计时工资

　　C 固定工资

　　D 浮动工资

　　答案　D

　　8、实行浮动工资对企业具有积极的作用，其中包括（ ）等。

　　A 不断提高工资水平

　　B 不断降低人员成本

　　C 克服平均主义，改善经营管理；

　　D 绩效与工资分离

　　答案　C

　　9、岗位技能工资制主要体现（ ）之间的差别。

　　A 不同劳动岗位

　　B 相同劳动岗位

　　C 不同劳动绩效

　　D 不同劳动者

　　答案　A

　　10、根据职工所在岗位的不同以及统一岗位内技术熟练程度的不同，划分等级，确定职工工资的制度称为（ ）。

　　A 浮动工资制

　　B 岗位技能工资制

　　C 固定工资制

　　D 提成工资制

　　答案　B

　　11、专家高薪制一般不受（ ）的限制

　　A 专家责任与业绩

　　B 企业实际需要

　　C 现行工资制度和水平

　　D 企业财力

　　答案　C

　　12、年薪制下经营者的工资单独考虑，经营者的收入将与（ ）挂钩。

　　A 职工工资水平

　　B 管理者的工龄

　　C 国有资产保值增值，企业规模、企业利润增幅

　　D 国家行政级别

　　答案　C

　　13、提成工资适合于哪些给企业直接带来经济效益的工作，如：（ ）。

　　A 董事工作

　　B 思想政治工作

　　C 企业文化建设

　　D 新产品开发与推广开发

　　答案　D

　　14、奖金是辅助的工资形式，是职工（ ）的报酬。

　　A 业余劳动

　　B 标准劳动

　　C 超额劳动

　　D 义务劳动

　　答案　C

　　15、奖金具有的特点是灵活性、针对性和（ ）。

　　A 全面性

　　B 及时性

　　C 平均性

　　D 刺激性

　　答案　B

　　16、企业主要的奖金形式有综合奖、超额奖和（　　）以及其他奖金形式等。

　　A 单项奖

　　B 集体奖

　　C 平均奖

　　D 福利

　　答案　A

　　17、职工全面完成各项技术经济指标所得的奖励应为（ ）。

　　A 单项奖

　　B 综合奖

　　C 福利

　　D 单项奖

　　答案　B

　　18、津贴是按劳分配的一种补充形式，在企业中应发给（ ）。

　　A 所有职工

　　B 所有干部

　　C 处于特殊劳动条件和环境的职工

　　D 有特殊贡献的人

　　答案　C

　　19、在企业中津贴与其他工资形式的关系是（ ）。

　　A 与奖金一样

　　B 与工资一样

　　C 与福利一样

　　D 与工资、奖金、福利不同

　　答案　D

　　20、在下列哪中情况下可以使用津贴（ ）？

　　A 普通标准工作岗位

　　B 劳动条件差，生活支出多；

　　C 超额完成任务

　　D 全勤

　　答案　B

　　（答案仅供参考）
