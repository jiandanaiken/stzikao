  • 09年自考“企业管理概论”练习题（13）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、（ ）是指产品、过程或服务满足规定和潜在要求（或需要）的特征和特性总和。

　　A　产品

　　B　质量

　　C　产品质量

　　D　性能

　　参考答案：　B

　　2、（ ）就是指产品满足使用者需要所具备的各种特性，也就是产品的使用价值。

　　A　产品

　　B　质量

　　C　产品质量

　　D　性能

　　参考答案：　C

　　3、产品质量有个产生、形成和实现的过程，这个过程中的各项工作活动的总称为（　　）。

　　A　产品质量

　　B　质量

　　C　质量职能

　　D　开发制造

　　参考答案：　C

　　4、质量职能的各项活动并不都是在一个企业的范围内进行的， 还涉及到企业外的（　　）。

　　A　供应商

　　B　用户

　　C　供应商和用户

　　D　政府

　　参考答案：　C

　　5、（　　）是全面质量管理的基本出发点。

　　A　全过程管理

　　B　全面管理

　　C　全员管理

　　D　满足用户需要

　　参考答案：　D

　　6、全"质"管理、全过程管理、全员管理、全方法管理、以及满足用户需要是（　　）特点。

　　A　全面质量管理

　　B　工作质量管理

　　C　产品质量管理

　　D　质量管理

　　参考答案：A

　　7、为使人们确信某一产品、过程、服务质量能满足规定的质量要求所必须的有计划的、有系统的全部活动。就是讲保证用户购得的产品在寿命期内质量可靠，这样的一系列工作称为（　　）。

　　A　质量控制

　　B　质量管理

　　C　质量体系

　　D　质量保证

　　参考答案：　D

　　8、对确定和达到质量要求所必需的职能和活动的管理称为（　　）

　　A　质量控制

　　B　质量管理

　　C　质量体系

　　D　质量保证

　　参考答案：　B

　　9、质量体系的内容包括（　　）

　　A　组织机构、职责和权限

　　B　组织机构、职责和权限、工作程序

　　C　组织机构、职责和权限、工作程序资源和人员、质量活动

　　D　质量活动、资源和人员

　　参考答案：　C

　　10、建立组织机构、明确职责和权限、工作程序、安排资源和人员，开展质量活动是（　　）内容

　　A　质量审核

　　B　质量小组

　　C　全面质量管理

　　D　质量体系

　　参考答案：　D

　　11、设计过程、生产制造、辅助生产和使用过程的管理过程是（　　）的全部质量活动。

　　A　质量体系中

　　B　全面质量管理

　　C　质量管理小组

　　D　质量保证

　　参考答案：A

　　12、加强工艺管理，做好组织技术检验、掌握质量动态，加强不合格品的管理、实行工序控制是（　　）的管理活动。

　　A　设计过程

　　B　生产制造

　　C　辅助生产

　　D　使用过程

　　参考答案：　B

　　13、制定质量方针和目标，是（　　）要抓的第一个工作。

　　A　建立和运行质量体系

　　B　全面质量管理

　　C　建立一套灵敏的质量信息反馈系统

　　D　质量保证

　　参考答案：A

　　14、分配和落实质量职能，完善质量文件；建立专职的质量管理机构；建立一套灵敏的质量信息反馈系统；按照管理循环不停顿，周而复始的运转，是（　　）需要抓的五个环节中的四个环节。

　　A　质量体系的建立和运行

　　B　全面质量管理

　　C　质量保证

　　D　质量审核

　　参考答案：A

　　15、制定质量目标、活动计划、管理项目和措施方案，是（　　）第一个阶段

　　A　建立一套灵敏的反馈系统

　　B　PDCA质量体系循环过程中的

　　C　全面质量管理

　　D　质量保证

　　参考答案：　B

　　16、分析现状，找出存在的质量问题；分析产生质量问题的各种原因和影响因素；从各种原因中找出影响质量的主要原因；针对影响质量的主要原因，制定技术组织措施方案。这是（　　）。

　　A　计划

　　B　实施

　　C　检查

　　D　处理

　　参考答案：A

　　17、在质量管理中需要利用数理统计方法，其工作程序分为（　　）个步骤。

　　A　一个

　　B　二个

　　C　三个

　　D　四个

　　参考答案：　C

　　18、按照不同的目的、标志进行分类，把性质相同、生产条件相同的数据归为一类，亦称

　　A　散布图法

　　B　排列图法

　　C　分类法

　　D　直方图法

　　参考答案：　C

　　19、分类法，是加工整理数据的一种重要方法，也是分析影响（　　）原因的一种基本方法。

　　A　工差

　　B　工序

　　C　质量

　　D　工种

　　参考答案：　C

　　20、（　　）是用于分析和判断工序是否处于稳定状态所使用的带有控制界限的一种图表。

　　A　鱼刺图

　　B　直方图

　　C　控制图

　　D　散布图

　　参考答案：　C

　　21、（　　），是利用各种统计报表，统计整理数据，对影响产品质量的原因做粗略分析。

　　A　鱼刺图法

　　B　统计分析表法

　　C　相关因素法

　　D　分类法

　　参考答案：　B

　　22、产品质量与影响产品质量因素之间，有一定的依存关系，但又不是一种严格的函数关系，这时可以做（　　）进行相关关系分析。

　　A　直方图

　　B　散布图

　　C　鱼刺图

　　D　排列图

　　参考答案：　B

　　23、（　　）年国际标准化组织发布了ISO9000系列标准。

　　A　1967

　　B　1977

　　C　1988

　　D　1987

　　参考答案：　D

　　24、ISO9000系列标准共分（　　）部分

　　A　2

　　B　3

　　C　4

　　D　5

　　参考答案：　D

　　25、ISO9000《质量管理和质量保证标准——选择和使用指南》，是该系列标准的（　　）。

　　A　选用导则

　　B　在合作环境，指导企业质量管理的标准

　　C　在非合作环境，指导企业质量管理的标准

　　D　一般说明

　　参考答案：A

　　26、《质量体系——生产、安装的质量保证模式》是（　　）的标题。

　　A　ISO9000

　　B　ISO9001

　　C　ISO9002

　　D　ISO9003

　　参考答案：　C

　　27、《质量体系——开发设计／开发、生产、安装和服务的质量保证模式》是（　　）的标题。

　　A　ISO9000

　　B　ISO9001

　　C　ISO9002

　　D　ISO9003

　　参考答案：　B

　　28、质量体系认证程序是（　　）。

　　A　申请——检查与评定——监督管理

　　B　申请——审批注册发证——监督管理

　　C　申请——检查与评定——审批注册发证——监督管理

　　D　申请——检查与评定

　　参考答案：　C

　　29、企业获得质量体系评审合格证书可以增强客户与供应者之间的信任感。是（　　）作用之一。

　　A　质量保证

　　B　质量认证

　　C　质量审核

　　D　质量检验

　　参考答案：　B

　　30、产品质量和全面质量管理；质量体系；质量管理的统计方法，ISO9000系列标准是（　　）率的内容。

　　A　生产计划

　　B　全面质量管理

　　C　设备管理

　　D　劳动组织与劳动定额

　　参考答案：　B

　　31、产品质量、质量职能、全面质量管理是（　　）节的内容

　　A　产品质量和全面质量管理

　　B　质量体系

　　C　质量管理的统计方法

　　D　ISO9000系列标准。

　　参考答案：A

　　（答案仅供参考）
