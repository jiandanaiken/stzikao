　　35、激励的目的是（ ）。

　　A 调动人们的积极性

　　B 企业获得更大的经济效益

　　C 企业获得更大的社会效益

　　D 使职工更重视经济利益

　　答案　A

　　36、实现有效的指挥，必须建立（ ）。

　　A 以厂长（经理）为首的指挥系统

　　B 集权制度

　　C 分权制度

　　D 以职工意愿为主的系统

　　答案　A

　　37、指挥职能是（ ）其它职能的。

　　A 独立于

　　B 排斥

　　C 相融于

　　D 高于

　　答案　C

　　38、按照既定计划和其它标准对企业的生产经营活动进行监督、检查、 发现偏差，采取纠正措施，使工作按原定计划进行，或以达到预期目的的管理活动称为（ ）。

　　A 控制职能

　　B 指挥职能

　　C 激励职能

　　D 计划职能

　　答案　A

　　39、选择的最为关键的控制要点被称为（ ）。

　　A 有效控制点

　　B 质量控制点

　　C 关键控制点

　　D 长期控制点

　　答案　C

　　40、控制是企业（ ）每一个主管人员的职责。

　　A 以下皆为

　　B 中层

　　C 基层

　　D 高层

　　答案　A

　　41、企业是一个（ ）。

　　A 政治组织

　　B 社会组织

　　C 经济组织

　　D 福利组织

　　答案　C

　　42、企业满足社会需要，实现自身价值，必须以（ ）。

　　A 经营为中心

　　B 销售为中心

　　C 政治为中心

　　D 福利事业为中心

　　答案　A

　　43、企业作为一个经济组织，其特征是（ ）。

　　A 国家拨款，自主经营

　　B 独立自主，自负盈亏

　　C 附属于国家行政机构

　　D 独立自主，不计盈亏

　　答案　B

　　（答案仅供参考）
