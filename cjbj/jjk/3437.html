  • 09年自考“企业管理概论”综合题（7）
  • 发布日期时间：2009/4/6  来源：zikao365   点击数：  作者：本站整理

　　1、企业基础工作主要是为实现企业目标和管理职能提供（ ）等，是一项必不可少的工作。

　　A 资料依据、共同准则、基本手段和前提条件

　　B 市场信息

　　C 组织机构和指挥

　　D战略决策

　　答案　A

　　2、企业管理基础工作的特点是（ ）。

　　A 一般性、群众性

　　B 阶段性、暂时性

　　C 科学性、群众性、先行性

　　D 随机性、先行性、个别性

　　答案　C

　　3、随着企业管理现代化的发展，企业管理基础工作的地位和作用是（ ）。

　　A 逐渐消失

　　B 有所下降

　　C 已经改变

　　D日趋重要

　　答案　D

　　4、企业基础工作在生产经营活动中是（ ）。

　　A 计划、组织和控制的依据

　　B 工作标准

　　C 计划、组织和控制的过程

　　D 工作要求

　　答案　A

　　5、企业基础工作是建立正常的生产秩序、提高生产效率和产品质量的（ ）。

　　A 重要决策

　　B 组织指挥

　　C 重要手段

　　D 监督保证

　　答案　C

　　6、按照时代特点划分，企业管理方法有（ ）。

　　A 经济方法和行政方法

　　B 定性方法和定量方法

　　C 传统方法和现代方法

　　D法律方法和行政方法

　　答案　C

　　7、按照解决问题的思维方式划分，企业管理的方法可以分为（ ）。

　　A 经济方法和行政方法

　　B 定性方法和定量方法

　　C 传统方法和现代方法

　　D法律方法和行政方法

　　答案　B

　　8、随着企业管理现代化的发展，未来的趋势是（ ）。

　　A 定量方法取代定性方法

　　B 定性方法取代定量方法

　　C 定量与定性同等重要

　　D 产生取代着两种方法的新方法

　　答案　C

　　9、企业管理的基础工作要求树立（ ）的管理观念。

　　A 尊重科学，按客观规律办事

　　B 讲究效率，压低成本

　　C 领导先行，重点突破

　　D创新、发挥

　　答案　A

　　10、做好企业的基础工作应（ ）。

　　A 立足当前，重点突破

　　B 长远考虑，系统设计

　　C 各自为政，独立完成

　　D 阶段进行，渐进突破

　　答案　B

　　11、标准化工作包括技术标准和管理标准的制定以及（ ）。

　　A 宣传工作

　　B 培训工作

　　C 研究工作

　　D执行和管理工作

　　答案　D

　　12、技术标准主要针对（ ）等。

　　A 生产对象和生产条件等

　　B 管理职责和程序等

　　C 人员规范和组织行为

　　D 产品质量和类别

　　答案　A

　　13、衡量一个国家和企业生产技术水平和管理水平的重要标志是（ ）。

　　A 企业规模

　　B 管理人员数量

　　C 标准化水平

　　D生产产量

　　答案　C

　　14、在推行标准化过程中，应逐步实行（ ）。

　　A 管理业务标准化

　　B 人员数量标准化

　　C 管理方式统一化

　　D 管理组织规模化

　　答案　A

　　15、用一种标准的单位量，去测定另一同类量的量值，称为（ ）。

　　A 数量方法

　　B 计量

　　C 勘测

　　D测量

　　答案　B

　　16、计量工作包括对各种理化性能测定与分析，以及（ ）等工作。

　　A 测试、检验

　　B 调查研究

　　C 汇总报表

　　D 宣传和贯彻

　　答案　A

　　17、计量工作的基本要求包括（ ）等。

　　A 按照自己规定办理

　　B 各部门兼职进行

　　C 计量器具准确可靠

　　D 企业领导亲自进行

　　答案　C

　　18、计量工作的基本要求包括（ ）等。

　　A 按照自己规定办理

　　B 各部门兼职进行

　　C 企业领导亲自进行

　　D 设置计量人员和机构

　　答案　D

　　19、企业信息工作包括原始记录、统计分析、技术经济情报和（ ）以及数据资料的收集、处理、传递、储存等管理工作。

　　A 科技档案

　　B 财务分析

　　C 市场调研

　　D产品创新

　　答案　A

　　20、记载企业生产技术经济活动情况的最初的直接纪录，称为（ ）。

　　A 统计工作

　　B 会计工作

　　C 计量工作

　　D 原始记录

　　答案　D

　　21、对原始纪录进行收集、整理和分析的工作称为（ ）。

　　A 会计工作

　　B 统计工作

　　C 审计工作

　　D 计量工作

　　答案　B

　　22、企业所需要的情报一般分成（ ）两类。

　　A 外部情报和内部情报

　　B 经济情报和科技情报

　　C 历史情报和当前情报

　　D加工情报和原始情报

　　答案　B

　　23、科技情报一般反映的是有关（ ）等的资料。

　　A 技术水平和技术潜力

　　B 外部环境和经营决策

　　C 产品数量和质量

　　D 组织机构和规章

　　答案　A

　　24、科技档案是企业的重要保存资料，他是在（ ）以及工程建设中形成的。

　　A 生产管理和市场开发

　　B 生产技术和科学研究

　　C 组织建设和人员配备

　　D产品销售和广告宣传

　　答案　B

　　25、科技档案工作包括档案的收集、整理、坚定、保管、统计和（ ）等六项工作。

　　A 复印

　　B 撰写

　　C 提供利用

　　D 公开传播

　　答案　C

　　26、科技档案管理在企业中应（ ）。

　　A 兼职进行

　　B 领导进行

　　C 专门机构或人员进行

　　D 分散管理

　　答案　C

　　27、企业在生产经营活动中所获得的有关数字的凭证，称为（ ）。

　　A 数据

　　B 消息

　　C 统计

　　D指示

　　答案　A

　　28、搞好数据管理应做到（ ）等要求。

　　A 数据分散

　　B 数据规模

　　C 数据统一

　　D 数据的存出

　　答案　C

　　29、制度化管理是以（ ）为主要手段协调企业组织集体协作行为的管理方式。

　　A 企业文化

　　B 物质利益

　　C 行政命令

　　D制度规范

　　答案　D

　　30、制度化管理的优越性体现在（ ）等方面。

　　A 个人与权力结合

　　B 责任和利益结合

　　C 个人与权力分离

　　D 适合小型企业

　　答案　C

　　31、制度化管理的实质是约束机制来自（ ）。

　　A 科学制定的制度规范

　　B 企业文化

　　C 承包责任制

　　D 领导的威信

　　答案　A

　　32、企业的规章制度体系由工作制度、责任制度和（ ）构成。

　　A 基本制度

　　B 考勤制度

　　C 分配制度

　　D学习制度

　　答案　A

　　33、在企业中工作制度和责任制度经常的存在方式是技术规范、业务规范、个人行为规范和（ ）。

　　A 语言规范

　　B 管理制度

　　C 工作任务

　　D 劳动定额

　　答案　B

　　34、决定企业性质，规定企业的形成和组织方式的制度是（ ）。

　　A 工作制度

　　B 责任制度

　　C 基本制度

　　D 学习制度

　　答案　C

　　35、在企业各项制度的基础层次的是（ ）。

　　A 基本制度

　　B 奖励制度

　　C 岗位责任制度

　　D学习制度

　　答案　C

　　36、岗位责任制度的内容一般包括（ ）。

　　A 岗位等级

　　B 岗位的职责

　　C 岗位的报酬

　　D 岗位的奖惩

　　答案　B

　　37、企业组织是由（ ）为主题建立的生物有机体。

　　A 信息

　　B 物资

　　C 人

　　D资金

　　答案　C
