　　1、生产计划的主要指标及编制，生产任务的统筹安排、设备负荷平衡核算是（ ）章的内容。

　　A 生产计划

　　B 设备管理

　　C 全面质量管理

　　D 生产作业计划与控制

　　参考答案：A

　　2、生产计划一章包括（ ）内容。

　　A 生产计划的主要指标及编制

　　B 编制生产作业计划的工作内容和作用

　　C 质量体系

　　D 设备的使用

　　参考答案：A

　　3、企业在计划期内生产可销售的产品的实物数量和工业性劳务的数量是指（ ）。

　　A 产品品种指标

　　B 产品质量指标

　　C 产品产量指标

　　D 产值指标

　　参考答案： C

　　4、企业在计划内应该生产的品种、规格的名称和数量指的是（ ）

　　A 产品品种指标

　　B 产品质量指标

　　C 产品产量指标

　　D 产值指标

　　参考答案：A

　　5、企业在计划期内生产可销售的各种产品应该达到的质量目标是指（ ）

　　A 产品品种指标

　　B 产品质量指标

　　C 产品产量指标

　　D 产值指标

　　参考答案： B

　　6、企业在计划期内应当出产可供销售的产品和工业性劳务的价值称为（ ）

　　A 总产值

　　B 商品产值

　　C 工业增加值

　　D 净产值

　　参考答案： B

　　7、用货币表现的企业在计划期内应该完成的工作总量称为（  ）。

　　A 总产值

　　B 商品产值

　　C 工业增加值

　　D 国民生产总值

　　参考答案：A

　　8、工业企业在计划期内以货币表现的工业生产活动的最终成果称为（  ）。

　　A 总产值

　　B 商品产值

　　C 工业增加值

　　D 国民生产总值

　　参考答案： C

　　9、工业（   ）包括成品生产价值，对外加工费收入自制半成品，在制品期末期初差额价值。

　　A 总产出

　　B 中间投入

　　C 增加值

　　D 总产值

　　参考答案：A

　　10、按收入法（分配法）计算工业增加值，公式为（  ）

　　A 固定资产折旧＋劳动者报酬

　　B 固定资产折旧＋劳动者报酬＋生产税净额

　　C 固定资产折旧＋劳动者报酬＋营业盈金

　　D 固定资产折旧＋劳动者报酬＋生产税净额＋营业盈金

　　参考答案： D

　　11、调查、掌握编制生产计划的依据；统筹安排；综合平衡，确定生产计划指标；报请上级主管部门批准或备案。是（  ）编制步骤。

　　A 劳动计划

　　B 生产计划

　　C 生产作业计划

　　D 车间作业计划

　　参考答案： B

　　12、生产计划工作属于计划管理的内容，必须贯彻（  ）原则。

　　A 投入产出

　　B 以销定产

　　C 量力而行

　　D 均衡生产

　　参考答案： B

　　13、企业计划明年生产某种产品，销售单价为500元，单位产品的变动成本为250元，预计明年总的固定成本为800万元，则界限产量为（   ）台。

　　A 32000

　　B 30000

　　C 36000

　　D 28000

　　参考答案：A

　　14、在上例中，企业还要求实现计划利润150万元，问明年计划产量应订多少（   ）台

　　A 42000

　　B 36000

　　C 40000

　　D 38000

　　参考答案： D

　　15、平均分配，分期递增，小幅度连续增长，抛物线形递增是产品各季、各月的市场需求量比较（   ），或者企业生产任务饱满时产品出产进度安排方式。

　　A 稳定

　　B 不稳定

　　C 季节性

　　D 常年性

　　参考答案：A

　　16、均衡安排方式，变动安排方式，折衷方式是产品需求具有（   ）的出产进度安排方式。

　　A 常年性

　　B 季节性

　　C 不确定性

　　D 不可预测性

　　参考答案： B

　　17、要先安排经常生产的和产量较大的产品；实行"集中轮番"的安排方式，新老产品交替要有一定的交叉时间；各类产品应合理搭配；应考虑各个品种的先后是有关（ ）应考虑的五个问题。

　　A 出产进度安排

　　B 产品产量优选

　　C 品种搭配

　　D 车间任务安排

　　参考答案： C

　　18、安排车间生产任务，应该实现（   ）要求。

　　A 必须保证整个企业的生产计划得到实现；缩短生产周期和减少流动资金的占用量。

　　B 必须保证整个企业的生产计划得以实现。

　　C 必须保证整个企业的生产计划得以实现，缩短生产周期和减少流动资金的占用量；充分利用车间的生产能力。

　　D 必须保证整个企业的生产计划得以实现；充分利用车间的生产能力。

　　参考答案： C

　　19、（   ），将企业的生产任务同企业现有机器设备能力进行核算对比。目的是检查生产能力对生产任务的保证程度。检查各季、各月设备的负荷是否均衡。

　　A 设备负荷平衡核算

　　B 设备的有效利用核算

　　C 开工设备的负荷计算

　　D 生产能力的计算

　　参考答案：A
