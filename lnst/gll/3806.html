浙江省2008年7月高等教育自学考试

　　工商行政管理学概论试题

　　课程代码：00108

　　一、单项选择题(本大题共20小题，每小题1分，共20分)

　　在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

　　1．工商行政管理的手段主要是( )

　　A.行政手段和经济手段 B.法律手段和教育手段

　　C.行政手段和法律手段 D.经济手段和法律手段

　　2．当需求缺乏弹性时，需求弹性系数为( )

　　A.大于1 B.等于1

　　C.小于1 D.无穷大

　　3．当市场供给和需求平衡时所达到的价格，通常称为( )

　　A.最高价格 B.最低价格

　　C.市场调节价格 D.市场均衡价格

　　4．作为工商行政管理法律法规的特殊的法律渊源是( )

　　A.法律解释 B.行政法规和行政规章

　　C.地方性法规和规章 D.宪法和法律

　　5．暂扣或者吊销执照是工商行政管理机关依法实施行政处罚中的( )

　　A.荣誉罚 B.行为罚

　　C.人格罚 D.财产罚

　　6．工商行政管理机关执法人员适用简易程序查处违法行为时，其执法身份证件( )

　　A.可以出示 B.应当出示

　　C.不必出示 D.可以不出示

　　7．根据工商行政管理机关的职责划分，受省级人民政府和国家工商行政管理局双重领导的是( )

　　A.省、自治区、直辖市工商行政管理局 B.国家工商行政管理局

　　C.市、县级工商行政管理局 D.工商行政管理所

　　8．市场经营主体的最本质最重要的特征是( )

　　A.竞争性 B.独立性

　　C.营利性 D.平等性

　　9．在新产品投放市场初期，采用的定价策略是( )

　　A.渗透定价策略 B.折扣定价策略

　　C.差别定价策略 D.撇脂定价策略

　　10．在下列各项中，不属于公用企业经营的行业是( )

　　A.食品 B.供电

　　C.邮政 D.供水

　　11．根据《担保法》规定，下列不适用于质押担保方式的财产是( )

　　A.存款单 B.房屋

　　C.股票 D.货物

　　12．向社会发布的声、形兼备，注目率高、广告效果强的广告是( )

　　A.报纸广告 B.杂志广告

　　C.电视广告 D.广播广告

　　13．发布特殊用途化妆品广告，必须提供( )

　　A.省级化妆品检测站（中心）或卫生防疫站出具的合格证明

　　B.地（市）级化妆品检测站（中心）或卫生防疫站出具的合格证明

　　C.市、县级化妆品检测站（中心）或卫生防疫站出具的合格证明

　　D.国务院卫生行政部门核发的批准文号

　　14．同一个商标所有人在相同商品上注册几个近似的商标或者同类的不同商品上注册几个相同或近似的商标，这类的商标称为( )

　　A.集体商标 B.联合商标

　　C.商品群商标 D.组合商标

　　15．商标自核准注册之日起，有效期为( )

　　A.10年 B.12年

　　C.15年 D.20年

　　16．国际消费者组织将“世界消费者权益日”定在每年的( )

　　A.3月15日 B.4月15日

　　C.10月15日 D.12月10日

　　17．有关维护消费交易公平的最典型的法律规范是( )

　　A.《民法通则》 B.《消费者权益保护法》

　　C.《反不正当竞争法》 D.《价格法》

　　18．二次大战后世界实施具有鲜明特色的社会市场经济体制的国家是( )

　　A.英国 B.德国

　　C.法国 D.意大利

　　19．下列各项中属于承诺的是( )

　　A.A商场向B制鞋厂发出定货单 B.顾客根据商场明码标价购买洗衣机

　　C.某上市公司的招股说明书 D.某家电超市海报公告有新款手机上市

　　20．商标最重要最本质的功能在于( )

　　A.区分不同商品生产者 B.便于广告宣传，鼓励正当竞争

　　C.获取最大利润 D.促进生产者努力提高商品质量

　　二、多项选择题(本大题共5小题，每小题2分，共10分)

　　在每小题列出的五个备选项中至少有两个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选、少选或未选均无分。

　　21．对商标注册进行实质审查，就是对商标注册申请条件的审查，包括审查商标的( )

　　A.合法性 B.显著性

　　C.实用性 D.鲜明性

　　E.新颖性

　　22．农民在购买，使用下列哪些农业生产资料时，可适用《消费者权益保护法》的有关规定( )

　　A.种子 B.农用汽车

　　C.农药 D.化肥

　　E.农用船舶

　　23．工商行政管理部门在查处下列哪些违法行为时可适用简易程序( )

　　A.退休工人王某无照经营

　　B.某公司擅自改变住所

　　C.个体工商户赵某出售猪肉短斤缺两，但情节轻微

　　D.某企业发布虚假广告

　　E.某商店出售盗版音像制品

　　24．按照《反不正当竞争法》规定，下列不属于不正当竞争行为的是( )

　　A.季节性降价 B.低价销售鲜活产品

　　C.假冒他人注册商标行为 D.低价处理有效期即将到期产品

　　E.因清偿债务、转产、歇业而降价销售商品

　　25．按照商品交换的方式来看，可以把市场分为( )

　　A.现货市场 B.国际市场

　　C.国内市场 D.期货市场

　　E.服务市场

　　三、填空题(本大题共5小题，每小题2分，共10分)

　　请在每小题的空格中填上正确答案。错填、不填均无分。

　　26．工商行政管理法律、法规的实施，就是指工商行政管理法律、法规的_____和_____。

　　27．根据《工商行政管理暂行规定》，对违法行为的查处，由违法_____或者违法_____所在地工商行政管理机关管辖。

　　28．市场经营主体与市场结构的关系主要取决于两个因素，即企业的_____和市场进入_____。

　　29．公司名称一般由行政区划、_____、行业特点和_____组成。

　　30．我国法律规定，禁止利用广播、电影、电视和报纸、期刊的_____，变相发布烟草广告等。烟草广告中必须标明“_____”字样。

　　四、名词解释(本大题共5小题，每小题3分，共15分)

　　31．市场失灵

　　32．商业秘密

　　33．不安抗辩权

　　34．广告管理

　　35．定金

　　五、简答题(本大题共5小题，每小题5分，共25分)

　　36．寡头垄断市场具有哪些特征？

　　37．对不正当竞争行为进行监督管理的原则是什么？

　　38．市场机制正常运行的基本条件有哪些？

　　39．注册商标撤销有哪些情形？

　　40．消费者组织的特征是什么？

　　六、论述题(本大题10分)

　　41．试论工商行政管理部门对消费者权益的保护。

　　七、案例题（本大题10分）

　　42．某市甲、乙、丙三家国有企业经市政府有关部门批准，共同出资组建一家以生产经营业务为主的有限责任公司。甲企业以货币出资，乙企业以厂房作价出资，丙企业以其工业产权出资，试问：

　　(1)该有限公司应在哪一级工商行政管理部门办理公司设立登记手续？领取什么执照？

　　(2)公司设立前，如何申办公司名称?

　　(3)该公司的注册资本和股东人数有什么规定？
