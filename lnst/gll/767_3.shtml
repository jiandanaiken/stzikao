C.沟通艺术                                     D.授权的艺术

C.控制标准和成效衡量

31.计划工作，就是根据实际情况，提出未来一定时期内组织所要达到的目标及实现目标的方法，具体内容包括(      )

A.做什么                                  B.何时做

C.何地做                                  D.何人做

E.如何做

32.下列各项中属于目标管理缺点的有(      )

A.目标管理理论还没有得到普及和宣传 

B.不利于暴露组织机构中的缺陷

C.目标一般是短期的，易造成短期目标与长期目标脱节

D.目标管理不够灵活

E.目标管理不利于进行有效的控制

33.人员配备原理包括(      )

A.责权一致原理                              B.灵活性原理

C.用人之长原理                              D.职务要求明确原理

E.集权与分权相结合原理

34.群众考评是对主管人员考评的一种方法，其中所谓的“群众”包括(      )

A.主管人员本人                              B.非主管人员

C.同级主管人员                              D.上级主管人员

E.下属主管人员

A.合格的主管人员所犯的错误最少

B.管理工作的成效是可以计量的

C.在计量管理工作成效时，管理的概念、原理和方法是有用的判别标准

D.出现的偏差可以预料并能及时发现
