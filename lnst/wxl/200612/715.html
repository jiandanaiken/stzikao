一、单项选择题(本大题共30小题，每小题1分，共30分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1.从发音方法看，“z、c”是(      )

A.塞音                             B.擦音

C.鼻音                             D.塞擦音

2.下列音节的韵母都是“ou”的是(      )

A.优、厚                           B.收、留

C.欧、洲                           D.求、救

3.音节“矿”、“物”各包含音素的数目是(      )

A.4个、2个                             B.3个、1个

C.3个、2个                             D.4个、1个

4.“身体力行”四音节单念时的调值分别是(      )

A.阴平、上声、去声、阳平               B.上声、阴平、去声、阳平

C.55、214、51、35                      D.214、55、51、35

5.音节“卧”、“室”的韵母分别属于四呼中的是(      )

A.合口呼、齐齿呼                     B.合口呼、开口呼

C.开口呼、齐齿呼                     D.开口呼、开口呼

6.下列词语中，属于单义词的是(      )

A.疙瘩                             B.伪装

C.通风                             D.贡品

7.合成词“扩大”的构成类型是(      )

A.联合式                             B.述宾式

C.偏正式                             D.补充式

8.下列词语中，字形完全正确的是(      )

A.变本加历                           B.一枕黄粮

C.按部就班                           D.走头无路

9.下列成语中，出自历史故事的是(      )

A.青梅竹马                           B.望梅止渴

C.成人之美                           D.七零八落

10.下列词语中，属于联合式合成词的是(      )

A.失望                               B.心酸

C.船只                               D.安危

11.下列语素中，都属于自由语素的一组是(      )

A.请、说                             B.目、宽

C.美、境                             D.居、阐

12.下列词语中，都带词辍的一组是(      )

A.布头、扇子                         B.强化、小楷

C.看头、演员                         D.读者、地头

13.下列词语中，词性相同的一组是(      )

A.现在、正在                         B.绿、绿化

C.野生、彩色                         D.从来、将来

14.下列短语中，不属于动词性短语的是(      )

A.那一次                             B.去问老师

C.希望参加                           D.不同意

15.下列句子中，述语后带宾语的是(      )

A.试验做得很成功。                   B.试验获得了成功。

C.她高兴得直笑。                     D.我吃过一回。

16.“这人好大的架子!”这个句子是(      )

A.名词谓语句                         B.动词谓语句

C.形容词谓语句                       D.主谓谓语句

17.下列切分中，第一层切分正确的是(      )

A.一只大白/公鸡                      B.妈妈/寄来的照片

C.高大的运动员的/衣服                D.一个非常/有趣的故事
