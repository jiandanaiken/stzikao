一、单项选择题（本大题共20小题，每小题1分，共20分）

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1.语言是人类借以思维和(       )

A.说话的工具                                         B.写文章的工具

C.彼此互相交际的工具                           D.彼此互相交际的最重要的工具

2.汉语方言中不是以地域命名的方言区是(       )

A.吴方言                                                B.闽方言

C.赣方言                                                D.客家方言

3.下列声母都是舌尖后音的一组字是(       )

A.出差省城                                            B.清扫操场

C.种植桑树                                            D.春蚕抽丝

4.下列韵母都是前鼻音的一组字是(       )

A.平民尽心                                            B.亲近贫民

C.心境清静                                            D.辛勤耕耘

5.下列注音正确的词语是(       )

A.现代(xiān dài)                                      B.汉语(hàn yǔ)

C.基础(jī chǚ)                                         D.课程(kè chén)

6.下列属于“词根+词缀(后缀)”的一组附加式合成词是(       )

A.感性、本性、重要性、可能性             B.弹性、变性、积极性、艺术性

C.记性、理性、创造性、先进性             D.党性、雄性、革命性、组织性

7.下列属于范围大小不同的一组同义词是(       )

A.阻止―制止―劝止                              B.边疆―边境―边界

C.要求―请求―恳求                              D.理想―幻想―空想

8.下列属于支配型结构的一组成语是(       )

A.饱经风雨、搜索枯肠、重整旗鼓

B.焕然一新、泥沙俱下、炎凉世态

C.宝山空回、史无前例、夜郎自大

D.知人善任、垂头丧气、迫不及待

9.“洗干净”是(       )

A.连动词组                                            B.递系词组

C.偏正词组                                            D.述补词组

10.“孩子呢?”是(       )

A.是非问句                                            B.特指问句

C.选择问句                                            D.反复问句

11.对“我们应该科学地分析科学的作用。”一句中的两个“科学”，下列判断正确的是(       )

A.都是状语                                            B.都是定语

C.前一个是状语，后一个是定语             D.前一个是定语，后一个是状语

12.复句关联词语“以便”表示(       )

A.连贯关系                                            B.条件关系

C.目的关系                                            D.假设关系

13.“你参加过‘结婚现场会’吗?大概没有，我参加过。”这个句子用于马烽某小说的开头，其作用是(       )

A.开门见山，突出全文中心                    B.问而不答，启发读者思考

C.问中有答，表示强烈语气                    D.引出下文，吸引读者注意

14.下列没有比喻义的词语是(       )

A.风云                                                   B.瓶颈

C.春蚕                                                   D.和稀泥

15.下列诗句“白发三千丈，缘愁似个长”的修辞方式是(       )

A.夸张                                                   B.比喻

C.比拟                                                   D.借代

16.下列“史”字前三画的笔画组合方式是(       )

A.相离                                                   B.相接

C.相连                                                   D.相交

17.下列都是并列结构的一组字是(       )

A.河边搭桥                                            B.祸起江湖

C.那棵松树                                            D.丽江新城

18.下列成语中没有错别字的是(       )

A.责无旁贷                                            B.情有独衷

C.一愁莫展                                            D.怨天由人

19.“刚才，四老爷和谁生气呢”这句话句末应加(       )

A.句号                                                   B.问号

C.叹号                                                   D.分号

20.下列连接号使用正确的是(       )

A.人类的发展可以分为古猿―猿人―古人―新人这四个阶段。

B.我国新开辟的北京――乌鲁木齐――贝尔格莱德航线首次通航。

C.马克・吐温（1835―1910），美国进步作家。

D.“中国～西方讨论会”将于明天举行。

二、多项选择题(本大题共5小题，每小题2分，共10分)

在每小题列出的五个备选项中至少有两个是符合题目要求的。请将其代码填写在题后的括号内。错选、多选、少选或未选均无分。

21.北京话里的“花儿”是（            ）

A.每个字独立发音形成的

B.在读“花”的韵母ua时，舌尖逐渐卷起来形成的

C.一个音节

D.两个音节

E.带有明显的卷舌色彩

22.“葡萄”是（            ）

A.一个语素                                            B.两个语素

C.词                                                       D.单纯词

E.词组

23.“他有两个弟弟：一个是记者，一个是编辑。”是（            ）

A.单句                                                   B.复句

C.并列复句                                            D.分合复句

E.注解复句

24.短句的特点有（            ）

A.简练明快                                            B.周全细致

C.感情色彩多样                                     D.信息量大

E.结构对称

25.汉字作为语素文字，其缺点有（            ）

A.由笔画构字                                         B.数量繁多

C.结构复杂                                            D.缺乏完备的表音系统

E.是形音义的统一体
