A.参差不齐                                                   B.饱经风霜

C.出水芙蓉                                                   D.踟蹰不前 

E.超然物外

14.属于构成成分部分相同的同义词是(    )

A.冲破――突破                                            B.怀念――思念

C.适合――合适                                            D.迫切――紧迫

E.夸奖――夸耀

15.下列句子中没有语病的是(    )

A.这句话可能被人误解。

B.这句话被人可能误解。

C.这句话被人没误解。

D.他对于我不理不睬，一副视若不见的样子。

E.他对我不理不睬，一副视若不见的样子。

16.“他唱了一首歌”是(    )

A.受事主语句                                         B.施事主语句

C.陈述句                                                D.单句

E.主谓句

17.“五岭逶迤腾细浪，乌蒙磅礴走泥丸”运用的修辞方式是(    )

A.排比                                                          B.双关 

C.夸张                                                          D.对偶

E.借代

18.“挨”是个多音字，正确的读音是(    )

A.āi                                                        B.ái 

C.ǎi                                                         D.ài

E.i

19.汉字笔画组合的方式有(    )

A.相离                                                          B.相接 

C.相合                                                          D.相交 

E.相包

20.下面例子中括号使用正确的有(    )

A.不久，国民议会迁到法皇的内宫凡尔赛去(在巴黎城西南十八里)。

B.为了保证教职员工正常洗澡，严禁带桶、盆入内(包括洗脸盆)。

C.我们的刊物(除了诗歌专刊)和报纸上的文字绝大多数是散文。

D.趁着今天党校开学的机会，我讲了这许多话，希望同志们加以考虑。(热烈的掌声)
