句内括号的后括号前面可以有句末点号

20.“我也知道你在想什么”这句话句末应加上(      )

A.句号                            B.问号

C.叹号                            D.分号

二、多项选择题(本大题共5小题，每小题2分，共10分)

在每小题列出的五个备选项中有二至五个是符合题目要求的，请将其代码填写在题干的括号内。错选、多选、少选或未选均无分。

21.韵母结构中充当韵头的元音有(          )

A.o                               B.i

C.u                               D.ü

E.e

22.下列各组字中，每个字的声母都是“f”的有(          )

A.蜂粉芬芳                         B.花粉丰富

C.非法发放                         D.凤凰翻飞

E.反复吩咐

23.“长篇大论”的反义成语有(          )

A.短小精捍                         B.精兵简政

C.言简意赅                         D.要言不烦

E.删繁就简

24.下列选项中属于词语巧妙配合的有(          )

A.无锡有锡了(新闻标题)

B.有的单位“门难进，脸难看，事难办”，官僚作风严重。

C.荆江分洪，舍小家保大家。

D.中“凤”洋“凰”韵味十足――中美演员同台演出《凤还巢》(新闻标题)

E.羊教授羊年话羊(新闻标题)

25.下列各组字中，每个字都是左右并列结构的有(          )

A.剑拔弩张                         B.江河海洋

C.激流彭湃                         D.烟波浩渺

E.报刊材料

三、名词解释(本大题共5小题，每小题3分，共15分)

26.汉语拼音方案

27.同音词

28.主谓谓语

29.比拟

30.错别字

四、改错说明题(本大题共3小题，每小题4分，共12分)

改正下列句子中的错误并扼要说明理由。

31.日本鬼子在华北、华南等地被打得身败名裂。

32.一个人呀，只有对党、对革命、对穷百姓充满无限的热爱，就没有什么困难不能克服，就没有什么事情不愿为之尽力。

33.他实事求是，不尚空谈，脚踏实地，任劳也任怨，总是把荣誉让给别人，把重担加于自己。

五、结构分析题(本大题共4小题，每小题4分，共16分)

用层次分析法分析下列句子的结构。

34.今年夏天热得不得了。

35.我希望你来一下儿。

36.他有事没有来。

37.我们大多数人的体内都潜藏着巨大的才能，但这种潜能酣睡着，一旦被激发，便能做出惊人的事业来。

六、辨析题(本大题共3小题，每小题4分，共12分)

38.选词填空，并作简要比较说明。

   当时他的病情已极为严重，全靠药物_____生命。(保持、维持)

39.比较“对”和“对于”的异同。

40.下面两句话在修辞方式上有何异同?

   (1)青蛙唱着恋歌，嫩蒲的香味散在春晚的暖气里。

   (2)我们要在人民中间生根、开花。

七、简答题(本大题共3小题，每小题5分，共15分)

41.“绿色食品、绿色能源、绿色材料”中的“绿色”表示什么意思?它和“绿的颜色”有什么词义关系?

42.简述“白”和“雪白”、“白茫茫”在意义和语法功能上的主要差别。

43.请将下列词组按语法规则归类。

   调查清楚            调查报告          调查嫌疑人

   在写                在书上写          写在书上

   学习文化            学习时间          研究报告
