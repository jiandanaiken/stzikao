一、单项选择题(本大题共20小题，每小题1分，共20分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1.普通话是指(      )

A.普普通通的话                      B.大家使用的共同语

C.汉民族使用的共同语                D.现代汉民族的共同语

2.“政治主张”这几个字的声母都是(      )

A.舌尖前音                          B.舌尖中音

C.舌尖后音                          D.舌面音

3.韵母都是后鼻音的一组字是(      )

A.清晨惊醒                          B.情境清静

C.经营金银                          D.进行定影

4.声调按“上声、去声、阴平、阳平”的顺序排列的一组字是(      )

A.雨后彩虹                          B.水远山高

C.柳暗花明                          D.水落石出

5.以音译方式吸收的一组外来词是(      )

A.模特儿、逻辑、比基尼              B.迷你裙、热狗、卡片

C.厄尔尼诺、沙发、啤酒              D.迪斯科、冰淇淋、可口可乐

6.结构方式都是复合式偏正型的一组词是(      )

A.铁路、牛肉、桃红                  B.证实、语言、椅子

C.简单、从前、花朵                  D.飞跑、体贴、人为

7.“做事很认真”是(      )

A.连动词组                          B.偏正词组

C.联合词组                          D.主谓词组

8.“这点心吃不得。”这个句子中的补语是(      )

A.结果补语                          B.状态补语

C.可能补语                          D.程度补语

9.“你想怎么办，我们就怎么办。”是(      )

A.对立关系复句                      B.倚变关系复句

C.递进关系复句                      D.条件关系复句

10.“天下没有不透风的墙。”是(      )

A.一般否定句，表意直率、明确        B.双重否定句，表示委婉的语气

C.一般不定句，表意委婉、平和        D.双重否定句，表示强烈的语气

11.“横眉冷对千夫指，俯首甘为孺子牛。”是对偶中的(      )

A.正对                              B.反对

C.串对                              D.侧对

12.“乱”字的最后一笔应归入五种基本笔形中的(      )

A.竖                                B.竖弯钩

C.钩                                D.折

13.按规范的书写顺序，“象”字的第五笔是(      )

A.竖                                B.横

C.撇                                D.折

14.下列汉字中，合体字是(      )

A.囚                                B.月

C.九                                D.册

15.下列成语中，有错别字的是(      )

A.再接再厉                          B.义不容辞

C.饮鸠止渴                          D.轻歌曼舞

16.下列句子中，问号使用有误的是(      )

A.“哎，你到底要不要?”“好，就来个五十斤吧?”

B.你说我这个体验对吗，同志?

C.你见过铁树开花吗?

D.看一个青年是不是革命的，拿什么做标准呢?

17.下列选项中，属于标号的是(      )

A.问号                             B.逗号

C.冒号                             D.括号

18.下列句子中，冒号起总结上文作用的是(      )

A.任弼时同志一生有三怕：一怕工作少，二怕麻烦人，三怕用钱多。

B.行动，要靠思想来指导；思想，要靠行动来证明：思想和行动是紧密联系的。

C.三七：中药名，即田三七。

D.女人抬头笑着问：“今天怎么回来得这么晚?”

19.下列说法中，对句内括号的用法解释不正确的是(      )

A.句内括号里的话是注释词语的

B.句内括号的后括号前面不能有点号

C.句内括号要紧贴在被注释者之后

D.
