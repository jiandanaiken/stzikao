一、单项选择题(本大题共20小题，每小题1分，共20分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1.思维活动的基本单位是(        )

A.概念                                                     B.分析

C.判断                                                     D.推理

2.2×1可以表示1个人手的数量，也可以是1双筷子的根数，它可以表示天地万物之间某一特定的数量关系，这表明数学学科具有(        )

A.抽象性                                                 B.系统性

C.具体性                                                 D.逻辑性

3.数学教育发展的总趋势是(        )

A.问题解决                                              B.一纲多本

C.编审分开                                              D.大众数学

4.数学作为一门课程列入我国中小学的教学计划始于(        )

A.元朝末年                                              B.明朝末年

C.清朝末年                                              D.建国初期

5.一年级学习10以内数的认识，学生通过数小棒、摆图片等认识了“几”和“第几”，这说明其思维正处于(        )

A.以直观行动思维为主                            B.以具体形象思维为主

C.以抽象逻辑思维为主                            D.以再造性思维为主

6.学生学习整数除法时，商是整数而余数为0，就叫除尽；继而学习小数除法，商是有限小数，也叫除尽。这是认知结构的(        )

A.同化过程                                              B.顺应过程

C.强化过程                                              D.迁移过程

7.数学操作技能的学习方法一般是(        )

A.引导发现法                                          B.范例学习法

C.尝试学习法                                          D.有意义学习法

8.学生掌握了直角、钝角、锐角的概念后，再把它们总括成“角”这种概念的同化是(        )

A.下位同化                                              B.类属同化

C.上位同化                                              D.并列同化

9.学生学习9+6=?时，默想其计算步骤和结果，这说明其智力技能的形成处于(        )

A.活动定向阶段                                       B.出声的外部言语阶段

C.不出声的外部言语阶段                         D.内部言语活动阶段

10.在教学过程中倡导“教师中心论”的教育家是(        )

A.杜威                                                     B.赫尔巴特

C.卢梭                                                     D.克伯屈

11.关于讲练课的基本结构，下列选项正确的是(        )

A.基本训练――引入新课――进行新课――尝试练习――阅读课本――独立练习――

布置作业

B.基本训练――引入新课――尝试练习――阅读课本――布置作业

C.引入新课――进行新课――练习――阅读――布置作业

D.基本训练――进行新课――布置作业――独立练习

12.适用于考查被试对概念细微的辨别能力、推理能力及运用原理解释问题的能力的题型是(        )

A.选择题                                                 B.是非题

C.改错题                                                 D.匹配题

13.根据解决数学问题的方向可把数学思维划分为(        )

A.形象思维与抽象思维                            B.逻辑思维与直觉思维

C.再造性思维与创造性思维                     D.集中思维与发散思维

14.学生对数学内容的好奇，对某门学科的感情都会激励他们积极地参加学习，这说明非智力因素对学生的学习具有(        )

A.定向作用                                              B.调节作用

C.维持作用                                              D.始动作用

15.下列因素中能够引起学生学习的外在动机的是(        )

A.兴趣                                                     B.爱好

C.荣誉                                                     D.求知欲

16.概念教学的中心环节是(        )

A.概念的引入                                          B.概念的理解

C.概念的巩固                                          D.概念的深化

17.“点”没有大小，“线”没有粗细，“面”没有厚薄，这种数学概念的抽象性表现是(        )

A.概括性                                                 B.原理性

C.理想化                                                 D.具体化

18.有目的，有顺序，持久的知觉(主要是视觉)活动是(        )

A.记忆                                                     B.观察

C.理解                                                     D.辨别

19.小学几何初步知识的性质是(        )

A.射影几何                                              B.抽象几何

C.直观几何                                              D.空间解析几何

20.学校教育、教学的主要形式是(        )

A.社会实践                                              B.课外活动

C.动手操作                                              D.课堂教学

二、多项选择题（本大题共15小题，每小题1分，共15分）

在每小题列出的五个备选项中有二个至五个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选、少选或未选均无分。

21.数学萌芽时期的运算主要有(          )

A.自然数的运算                                       B.整数的运算

C.小数的运算                                          D.分数的运算

E.四则运算

22.选取小学数学教学内容的依据是(          )

A.适合小学生的接受能力

B.选取与科技发展水平相一致的数学知识

C.根据九年义务教育的学制设置教学内容

D.根据小学课程计划设置教学内容

E.选择现代生活和进一步学习所必需的最基础的数学知识

23.单一式数学教材体系的主要内容有(          )

A.正整数                                                 B.正小数

C.正分数                                                 D.代数

E.几何

24.小学数学学习过程包括(          )

A.动机的激发                                          B.知识的感知

C.知识的理解                                          D.知识的巩固

E.知识的应用

25.下列属于智力技能的有(          )

A.尺规作图                                              B.测量教室面积

C.计算贷款利息                                       D.列方程解应用题

E.数字的书写

26.数学规则包括(          )

A.数学概念                                              B.数学法则

C.数学定律                                              D.数学术语

E.数学公式

27.形象思维的基本形式是(          )

A.表象                                                     B.直感

C.想像                                                     D.灵感

E.推理

28.根据不同的教学目的，设问的种类主要有(          )

A.引入新课时的设问                                B.理解新知识的设问

C.巩固新知识时的设问                            D.提高学生注意力的设问

E.应用知识时的设问

29.按考评的性质可将考评分为(          )

A.显示性考评                                          B.预示性考评

C.形成性考评                                          D.总结性考评

E.诊断性考评

30.下列属于小学数学思维的基本方法的是(          )

A.观察                                                     B.比较

C.分类                                                     D.分析

E.抽象

31.下列数学概念属于交叉关系的有(          )

A.质数与合数                                          B.自然数与整数

C.等腰三角形与直角三角形                     D.质数与奇数

E.真分数与假分数

32.注意是心理活动对一定事物的(          )

A.理解                                                     B.指向

C.识记                                                     D.反映

E.集中

33.下列所下定义不正确的是(          )

A.两直线相互垂直所构成的角叫直角

B.分数不是整数

C.像笔直的铁轨那样两条永不相交的直线叫平行线

D.分子小于分母的分数叫真分数

E.含有未知数的等式叫方程

34.造成计算错误，学生心理方面的原因表现为(          )

A.感知比较粗略                                B.情感比较脆弱

C.注意不够稳定                                D.不良学习习惯

E.知识掌握不牢

35.数学教师的语言具有

A.准确性                                                 B.科学性

C.逻辑性                                                 D.系统性      

E.概括性

三、名词解释题（本大题共4小题，每小题3分，共12分）

36.心理定势

37.问题解决

38.发现法

39.教学评价

四、简答题（本大题共4小题，每小题5分，共20分）

40.通过小学数学教学应培养小学生哪些数学能力？

41.联系教学实际谈谈如何贯彻传授数学知识和培养数学能力相结合的原则？

42.简述如何培养学生初步的逻辑思维能力。

43.简述小学数学教师的业务能力素质包括的主要内容。

五、论述题（本大题共2小题，每小题10分，共20分）
