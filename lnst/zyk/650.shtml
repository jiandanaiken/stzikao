一、单项选择题(本大题共20小题，每小题1分，共20分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1．人的专门化运动知觉的发展水平不同主要是由于_______差异造成的。(     )

A．分析器系统机能水平                          B．运动水平

C．个体差异                                           D．训练水平       

2．动作记忆的遗忘量比语词记忆的遗忘量要(     )

A．差不多                                               B．大

C．少                                                      D．目前不清楚                         

3．游泳项目属于(     )

A．开放性运动技能项目                          B．闭锁性运动技能项目

C．精细的运动技能项目                          D．粗大的技能项目

4．更有利于闭锁性运动技能形成的方法是(     )

A．分解性                                               B．完整法

C．直观性                                               D．示范性

5．一般来说，学习运动技能的方法中，哪一种练习方法效果好？(     )

A．间时练习                                           B．集中练习

C．不一定                                               D．完整练习

6．斯金纳认为_______是学习的最根本规律。(     )

A．情景反应                                           B．观察学习

C．强化                                                  D．顿悟

7．属于体育教学中的非言语指导的方式是(     )

A．启发                                                  B．教育

C．提示                                                  D．眼神

8．应具备较高唤醒水平的运动项目是(     )

A．耐力项目                                           B．游泳

C．体操                                                  D．技巧

9．运动焦虑在比赛前一周会达到很高水平，进入比赛后，焦虑会(     )

A．上升                                                  B．下降

C．不变                                                  D．不确定

10．最早的认知学习理论――格式塔心理学说的关键词是(     )

A．观察                                                  B．强化

C．反应                                                  D．顿悟

11．_______的学习理论揭示了人类普遍存在的学习方式，即观察学习理论。(     )

A．斯金纳                                               B．鲍尔顿

C．班杜拉                                               D．力克斯

12．体育运动中的特殊体验者有高峰表现、流畅体验与(     )

A．跑步者高潮                                        B．体育锻炼快感

C．情绪高潮                                           D．都不对

13．体育团队的凝聚力可分为任务凝聚力与(     )

A．交往凝聚力                                        B．方向凝聚力

C．团结凝聚力                                        D．竞争凝聚力

14．有关攻击性行为的理论学派有本能论、挫折――攻击论与(     )

A．个体学习理论                                    B．社会学习理论

C．需要论                                               D．人的本质论

15．原始体育的产生与发展是基于人的劳动与(     )

A．精神需要                                           B．文化需要

C．心理、生理需要                                 D．人的本质

16．影响体育团队凝聚力的因素主要有团队成员的相似性，团队内部协作，_______与团队间的竞争。(     )

A．团队成员的数量                                 B．个别差异
