  • 发布日期时间：2006-12-7  来源：自考365   点击数：  作者：佚名

一、填空题(本大题共4小题，每空1分，共10分)

请在每小题的空格中填上正确答案。错填、不填均无分。

1．学习由3个基本环节构成，它们是_______、_______和_______。

2．知觉的组织性主要体现在_______、_______和_______3条原则。

3．_______是态度形成的基础，_______是态度的核心。

4．客人投诉的心理原因主要有三个方面：（1）求尊重，（2）_______，（3）_______。

二、单项选择题(本大题共10小题，每小题2分，共20分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1．客人来到餐厅刚坐下，还没来得及看一遍菜单，服务员就要为客人点菜，导致客人不悦。这说明服务员没有能够(     )

A．随机应变                        B．掌握灵活性

C．掌握服务时机                        D．提供针对性服务

2．有的人嘴上说不喜欢杭州，却常常到杭州来旅游。这说明(     )

A．人的心理是多变的                        B．人的态度具有不一致性

C．态度具有不稳定性                        D．有些人心理不健康

3．以下4个选项中有一项不属于内容型激励理论。请指出(     )

A．马斯洛的“需求层次理论”

B．奥德费的“E・R・G理论”

C．亚当斯的“公平理论”

D．麦克利兰的“成就需要理论”

4．教材中指出，为了进一步完善企业形象，吸引更多的回头客，在服务的终结阶段可采取三种对策。指出以下选项中教材里没有提及的对策(     )

A．与客人说好最后一句话                        B．及时地帮助客人结帐

C．灵活机动地向客人道别                        D．认真做好善后工作

5．关于旅游者态度的有关理论，以下有四种说法。其中有一项是错误的。请指出(     )

A．态度是由认知、情感、意向三种成分所构成的内在心理倾向

B．旅游者态度具有“稳定性和可变性”双重特征

C．一个对旅游持不赞成态度的人就不会同意他的孩子报考这个专业。

D．包含着大量情绪性感受的创伤性经历会使人们的态度发生重大改变

6．以下关于心理的实质的说法中，有一项是错误的。请指出(     )

A．人的有些心理现象不是后天习得的，而是遗传就有的。如母爱

B．没有客观现实，就不能产生心理活动

C．没有人脑也不会有人的心理

D．神经病人和正常人一样，也有心理活动

7．关于马斯洛需要层次理论的理解，以下四项中有一项是不正确的。请指出(     )

A．人的高级需要是以人的低级需要的满足为前提的

B．人的基本需要只有“安全需要”和“生理需要”两项，其他三项是属于高级需要

C．低级需要可以通过外部条件得到满足

D．高级需要比低级需要对人更有激励作用

8．加拿大精神病医生埃里克・伯恩博士在他的专著《人们玩的游戏》形象地提出“父母”、“成人”、“儿童”三种自我状态。这个比喻是关于(     )

A．人格特征                        B．人格倾向

C．人格结构                        D．态度特征

9．有关职工群体动力的理论，以下哪种说法是不正确的？(     )

A．群体压力越大，对员工的激励作用就越强

B．群体的士气高，生产效率可能反而低

C．适当的群体竞争对企业发展是有必要的

D．群体压力是群体规范对其成员所具有的无形的约束力

10．以下哪一项不属于人的气质方面的内容？(     )

A．思维灵活敏捷                        B．注意力能够长时间集中于某件事

C．穿着整洁大方                        D．遇事不慌不乱

三、名词解释(本大题共5小题，每小题4分，共20分)

1．心理

2．知觉的解释性

3．旅游动机

4．气质

5．群体压力

四、简答题(本大题共5小题，每小题5分，共25分)

1．为什么说心理因素是旅游行为因素中的核心因素？

2．怎样理解单一性和多样性需要的平衡？旅游活动对调节人的生活在单一性和多样性需要之间的平衡有什么作用？

3．了解人格类型与旅游行为的关系，对开展旅游工作有什么现实意义？

4．社会阶层对人们的旅游行为的影响主要表现在哪几个方面？

5．思维的个别差异表现在哪几个方面？

五、论述题(本大题共2小题，共25分)

1．试解释一下确定领导风格的“连续统”理论。这一理论对我国的企业领导者有什么实际意义？（10分）

2．横向整体化服务包括哪些内容？结合实际谈谈如何应用所学心理学知识进行科学管理？
