一、单项选择题(本大题共20小题，每小题2分，共40分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1.以游戏为基本活动，根据教师对幼儿活动的干预、控制的程度差异，幼儿园的教育活动可以划分为(      )

A.自由游戏、活动区游戏和全班游戏

B.自由游戏、任务定向的活动区游戏和全班集体游戏

C.自由游戏、小组游戏和全班游戏

D.自由游戏和教师干预的游戏

2.根据学习过程中师生的互动关系，操作学习可分为(      )

A.师生合作性操作学习、个体探究性操作学习和群体协作性操作学习

B.示范性操作学习、指导性操作学习、个体探究性操作学习和群体协作性操作学习

C.师生合作性操作学习、示范性操作学习和指导性操作学习

D.指导性操作学习、个体探究性操作学习和群体协作性操作学习

3.安德森提出的对应于个人发展类教育活动的指导策略为(      )

A.小组调查和角色扮演                                  B.导学和模拟

C.非指导性咨询和综合                                  D.先行组织者和概念获得

4.亲子互动结构一般可以划分为以下三个层次(      )

A.爱的基础、成长的喜悦和关爱的需要

B.关爱的需要、成长的喜悦和健全的教育

C.爱的基础、成长的喜悦和健全的教育

D.爱的基础、成长的喜悦和健康的教育

5.布鲁纳认为，在人类智慧生长期间，有以下几种表征系统在起作用(      )

A.动作表征、肖像表征和符号表征                

B.动作表征、形像表征和符号表征

C.动作表征、肖像表征和语言表征                

D.动作表征、肖像表征和观念表征

6.幼儿在社会实践活动中获得社会性经验并改变其行为的过程就是(      )

A.幼儿社会性                                                B.幼儿社会性学习

C.幼儿社会化                                                D.幼儿社会学习

7.创设背景型教育活动情境应充分考虑教育活动的(      )

A.目标、空间、时间、内容、形式与气氛等要素

B.目标、空间、时间、内容、材料与形式等要素

C.空间、时间、内容、材料、形式与气氛等要素

D.目标、时间、内容、材料、形式与气氛等要素

8.加涅的学习记忆模型认为，学习过程是由以下三个系统协调活动的过程(      )

A.加工、预期、执行控制                               B.信息加工、信息执行和信息控制

C.感觉记忆、短时记忆和长时记忆                 D.反应、加工和执行

9.根据概念属性的抽象与概括的程度，可以把概念分为(      )

A.初级概念和二级概念                                  B.具体概念和抽象概念

C.具体概念和概括概念                                  D.前科学概念和科学概念

10.在教育活动进行中实施的评价叫(      )

A.发展性评价                                                B.过程性评价

C.形成性评价                                                D.诊断性评价

11.构成师幼互动的内隐心理要素有(      )

A.认知发展、角色认知、行为期待                 B.心理成熟、认知发展、行为期待

C.场景界定、角色认知、行为期待                 D.场景界定、角色扮演、行为期待

12.分析问题阶段中的创造性体现为(      )

A.阐述问题的能力                                         B.组织问题的能力

C.发现问题的能力                                         D.明确问题的能力

13.幼儿教育心理学研究的基本方法是(      )

A.实验法、观察法、调查法和测量法             B.观察法、实验法、调查法和问卷法

C.观察法、问卷法、实验法和测量法             D.实验法、测量法、问卷法和调查法

14.美国学者普林格尔认为，儿童的基本感情需要是(      )

A.对爱的需要、对新体验的需要、对赞扬和认可的需要、对责任感的需要

B.对爱的需要、对安全的需要、对赞扬和认可的需要、对责任感的需要

C.对爱和安全的需要、对新体验的需要、对赞扬和认可的需要、对责任感的需要

D.对爱的需要、对安全的需要、对赞扬的需要、对认可的需要

15.心理学研究发现，父母在教养模式上有两个重要的行为维度，其中一个是情绪维度，具体表现为(      )

A.过度限制、要求－放纵、不要求                 B.关心、热爱－不关心、冷漠

C.关注、热爱－放任、冷漠                           D.温暖、接纳－敌意、拒绝

16.“沃斯克－戴开放型与传统型学习环境及儿童课堂行为测定量表”对学习环境的评价主要包括以下几个变量(      )

A.场所、小组领导人和组内人数                    B.场所、小组领导人和学习氛围

C.场所、学习氛围和组内人数                        D.自然环境、学习条件和组内人数

17.在3－6或7岁的幼儿期，占有重要地位的、主要的游戏类型是(      )

A.结构游戏                                                    B.规则游戏

C.练习性游戏                                                D.象征性游戏

18.幼儿用新的方式进行某项游戏是属于(      )
