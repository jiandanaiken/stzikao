一、名词解释(任选做5小题，全做只给前5题分。每小题3分，共15分)

1.限制氨基酸

2.酶促褐变

3.糖元生成作用

4.蛋白质变性作用

5.改性纤维素

6.乳化剂

二、填空题(每空1分，共15分)

1.生物体系中能量传递体是含___________的化合物。

2.氨基酸的降解有两种基本方法：___________和___________。

3.蛋白质营养质量取决于它的___________组成与比例。

4.目前已知非酶褐变的机制有：羰氨反应褐变作用、___________和___________作用。

5.酚类色素可以分为花青素、花黄素和___________三大类。

6.水的生理功能表现在___________、___________、___________和水是体内摩擦的润滑剂。

7.复合型酶的结构中，除了有酶蛋白外，还有两种辅助因子，它们是___________和___________。

8.维生素A，又称为___________维生素，如果缺乏会导致人体出现___________。

9.淀粉糊化的温度一般在___________。

三、单项选择题(在每小题的四个备选答案中，选出一个正确答案，并将正确答案的序号填在题干的括号内。每小题1分，共15分)

1.在各种细胞色素中，只有(      )能直接以分子氧为受氢体。

A.细胞色素aa3                                                B.细胞色素b

C.细胞色素c                                                  D.细胞色素c1

2.下列物质中，不是TCA循环中间产物的是(      )。

A.乙醛酸                                                               B.苹果酸

C.柠檬酸                                                        D.琥珀酸

3.HMS途径又称(      )。

A.乙醛酸循环                                                        B.己糖单磷酸支路

C.三羧酸循环                                                        D.糖酵解

4.营养物质最主要的吸收部位在(      )。

A.口腔                                                            B.胃

C.小肠                                                            D.大肠

5.充分发挥蛋白质的(      )可以改善蛋白质营养价值。

A.保护作用                                                     B.互补作用

C.庇护作用                                                     D.抗生酮作用

6.从化学结构类型来看，血红素是(      )。

A.吡咯色素                                                     B.多烯色素

C.酚类色素                                                     D.吡啶色素

7.下列物质中，属于有毒植物蛋白质物质的是(      )。

A.草酸                                                            B.硫苷

C.棉酚                                                            D.凝集素

8.能够反映油脂新鲜程度的指标是(      )。

A.酸价                                                            B.碘价

C.过氧化价                                                     D.皂化价

9.糊化前的淀粉主要是(      )。

A.α-淀粉                                                       B.β-淀粉

C.γ-淀粉                                                       D.δ-淀粉

10.当淀粉的聚合度n=20~30时，与碘反应呈(      )。

A.蓝色                                                            B.无色

C.紫红色                                                               D.绿色

11.每克球状蛋白质一般可与(      )克水结合。

A.0.2~0.3                                                        B.0.3~0.4

C.0.4~0.5                                                               D.0.5~0.6

12.在酶反应中，Km被称为(      )。

A.米氏常数                                                     B.阿佛加德罗常数

C.阿莱尼乌斯常数                                                 D.范德华常数

14.脂肪变质的过程称为(      )。

A.腐败                                                            B.酸败

C.发酵                                                            D.失活

15.矿物质元素含量在(      )以上的为常量元素。

A.0.04%                                                          B.0.03%

C.0.02%                                                          D.0.01%

四、判断题(判断下列各题，正确的在题后括号内打“√”，错的打“×”。每小题1分，共15分)

1.正常情况下，生物体系内物质氧化总是与还原偶联进行的。(      )

2.高能磷酸化合物就是指三磷酸腺苷，也即ATP。(      )

3.蛋白质可以用作能源，一般正常情况下，在食物提供的总热量中的30%是由蛋白质提供的。

(      )

4.胡萝卜素和叶绿素是重要的两类吡咯色素。(      )

5.物质产生味感的前提是必须溶解于水。(      )

6.棉酚是有显著毒性的天然酚类物质，其毒性可以用湿热处理法予以消除。(      )

7.氨基酸具有酸、甜、苦、鲜等多种味道。(      )

8.蜡也是一种酯，在体内容易被消化吸收，具有一定的营养价值。(      )

9.能够阻止、延迟脂肪自动氧化作用的物质称为助氧剂。(      )

10.蛋白质、脂肪、碳水化合物的化学元素组成一样，所以生理作用也可以相互代替。(      )

11.糖酸可以根据其化学结构分为糖一酸、糖二酸和糖三酸。(      )

12.蔗糖、麦芽糖都是双糖，而且都属于还原性糖。(      )

13.液态的油是由不饱和脂肪酸组成的；固态的脂是由饱和脂肪酸组成的。(      )

14.酶元是酶的前体，与酶一样具有明显的催化功能。(      )

15.维生素B1，又称为核黄素。(      )

五、简答题(每小题5分，共30分)

1.生物氧化中高能磷酸键的产生方式有哪些？

2.脂肪酸的降解途径有哪几种？

3.水果、蔬菜成熟过程中会发生哪些生物化学变化？

4.抗氧剂的抗氧机理是什么？（用模式表示）

5.酶学中“诱导-契合理论”的内容是什么？

6.影响淀粉老化的因素有哪些？

六、论述题(任选做一题，全做只给第1题分。10分)

1.试述羰氨反应的过程。

2.论述蛋白质一级结构的内容。
