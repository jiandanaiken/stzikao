  • 全国2006年1月自学考试毛泽东思想概论试题
  • 发布日期时间：2006/7/2  来源：中国教育在线   点击数：  作者：未知

一、单项选择题(本大题共30小题。每小题1分，共30分)

在每小题列出的四个备选项中只有一个是符合题目要求的，请将其代码填写在题后的括号内。错选、多选或未选均无分。

1．中国共产党首次确立毛泽东思想作为党的指导思想的会议是(          )
A．遵义会议 B．瓦窑堡会议
C．中共七大 D．中共八大

2．毛泽东思想开始萌芽是在(          )
A．中国共产党创建和国民革命时期
B．土地革命战争前中期
C．土地革命战争后期和抗日战争时期
D．解放战争时期

3．毛泽东的《反对本本主义》一文所反对的错误倾向是(          )
A．投降主义 B．经验主义
C．教条主义 D．官僚主义

4．毛泽东思想的精髓是(          )
A．实事求是 B．群众路线
C．独立自主 D．艰苦奋斗

5．中国新民主主义革命始于(          )
A．中国无产阶级的产生 B．新文化运动的兴起
C．五四运动的爆发 D．中国共产党的成立

6．中国新民主主义革命胜利后建立的社会制度是(          )
A．资本主义制度  B．新民主主义制度
C．社会主义制度  D．共产主义制度

7．毛泽东明确地把官僚资本主义列为革命对象之一的著作是(          )
A．《中国社会各阶级的分析》 B．《新民主主义论》
c．《论联合政府》        D．《在晋绥干部会议上的讲话》

8．中国共产党在创建初期，把主要精力放在(          )
A．组织农民运动  B．发展工人运动
C．开展土地革命  D．领导军事斗争

9．“工农武装割据”的中心内容是(          )
A．武装斗争          B．土地革命
C．根据地建设    D．党的领导

10．赞成毛泽东农村包围城市的观点，提出“先有农村红军，后有城市政权”论断的领导人是(          )
A．周恩来 B．刘少奇
C．朱德  D．王稼祥

11．毛泽东关于人民战争战略战术思想的核心是(          )
A．战略上藐视敌人，战术上重视敌人的方针
B．战略进攻和战略决战的思想
C．集中优势兵力打歼灭战的方法
D．积极防御的思想

12．中国革命统一战线的基础是(          )
A．坚持无产阶级领导权 B．制定正确的政治纲领
C．建立巩固的工农联盟 D．团结一切可以团结的力量

13．在抗日民族统一战线中，中国共产党对顽固势力的策略方针是(          )
A．消灭顽固势力  B．孤立顽固势力
C．中立顽固势力  D．排斥顽固势力

14．无产阶级在革命统一战线中的领导权，主要是指中国共产党对于全国人民和各革命阶级的(          )
A．思想领导 B．组织领导
C．政治领导 D．军事领导

15．毛泽东第一次明确使用“人民民主专政”概念的文章是(          )
A．《新民主主义论》 B．《论联合政府》
C．《将革命进行到底》 D．《论人民民主专政》

16．我国人民民主专政政权的组织形式即政体是(          )
A．参议会制度    B．政治协商制度
C．多党合作制度  D．人民代表大会制度

17．我国建立的第一个民族自治区是(          )
A．内蒙古自治区  B．西藏自治区
C．宁夏回族自治区 D．新疆维吾尔自治区

18．在中国特色的社会主义政党制度中，民主党派是(          )
A．在野党 B．执政党
C．参政党 D．反对党

19．新中国成立初期，国营经济建立的最主要途径和手段是(          )
A．没收帝国主义在华企业   B．没收官僚资本
C．没收民族资本           D．没收地主阶级的土地和财产

20．1951年至1952年，我国开展的“三反”运动是(          )
A．反贪污、反浪费、反官僚主义
B．反行贿、反偷税漏税、反盗窃国家经济情报
C．反浪费、反盗骗国家财产、反偷工减料
D．反腐败、反浪费、反官僚主义

21．中国共产党提出的过渡时期总路线的主体是(          )
A．对农业的社会主义改造
B．对手工业的社会主义改造
C．对资本主义工商业的社会主义改造
D．实现国家的社会主义工业化

22．我国对资本主义工商业实行社会主义改造所采取的基本政策是(          )
A．加工订货 B．统购包销
C．和平赎买 D．收归国有

23．我国第一个五年计划提前完成的时间是(          )
A．1955年 B．1956年
C．1957年 D．1958年

24．毛泽东在《论十大关系》中提出的中国社会主义建设的基本方针是(          )
A．不要四面出击
B．积极引导，稳步前进
C．调动国内外一切积极因素为社会主义事业服务
D．统筹兼顾，全面安排

25．毛泽东指出，社会主义社会基本矛盾运动的特点是(          )
A．互相矛盾 B．互相适应
C．互相排斥 D．又相适应又相矛盾

26．董必武在中共八大上指出，加强人民民主法制的中心环节是(          )
A．有法可依 B．依法办事
C．执法必严 D．违法必纠

27．毛泽东指出，社会主义国家政治生活的主题是(          )
A．正确处理社会基本矛盾 B．正确处理两类社会矛盾
C．正确处理敌我矛盾 D．正确处理人民内部矛盾

28．邓小平在谈到中国共产党的建党学说时指出，把列宁的建党学说发展得最完备的是(          )
A．毛泽东 B．刘少奇
C．周恩来 D．任弼时

29．中国共产党内产生“左”右倾机会主义错误的主要思想根源是(          )
A．资产阶级思想 B．小资产阶级思想
C．封建主义思想 D．自由主义思想

30．中国共产党实行民主集中制的目的是(          )
A．加强党内的民主监督 B．提高党的执政能力
C．消除党内的官僚主义 D．保证党在思想上和组织上的一致

二、多项选择题(本大题共10小题，每小题2分，共20分)
在每小题列出的五个备选项中至少有两个是符合题目要求的。请将其代码填写在题后的括号内。错选、多选、少选或未选均无分。
31．在贯彻群众路线的过程中，必须实行(            )
A．集体与个人相结合
B．主观与客观相结合
C．领导与群众相结合
D．一般号召与个别指导相结合
E．民主基础上的集中和集中指导下的民主相结合

32．新民主主义革命的动力包括(            )
A．工人阶级      B．农民阶级
C．城市小资产阶级 D．民族资产阶级
E．大地主大资产阶级

33．在准备秋收起义时，毛泽东提出革命的目的是(            )
A．巩固和加强统一战线         B．创建和扩大农村革命根据地
C．坚持党对军队的绝对领导 D．夺取政权
E．解决农民的土地问题

34．我国倡导并得到世界广泛认同的处理国家之间关系的基本原则是(            )
A．互相尊重主权和领土完整 B．互不侵犯
C．互不干涉内政             D．平等互利
E．和平共处

35．毛泽东提出的中国共产党对民主党派实行的方针是(             )
A．长期共存 B．互相监督
C．肝胆相照 D．荣辱与共
E．团结互助

36．新中国建立初期，对帝国主义和其他资本主义国家在华企业逐步收归国有的形式有(             )
A．管制  B．限制
C．征用  D．代管
E．转让

37．新中国建立前夕,毛泽东进一步阐述农业合作化的必要性及重大意义的著作有(             )
A．《目前形势和我们的任务》
B．《新民主主义论》
C．《论人民民主专政》
D．《在晋绥干部会议上的讲话》
E．《在中共七届二中全会上的报告》

38．我国农业社会主义改造的步骤是建立(             )
A．带有社会主义萌芽性质的农业生产互助组
B．半社会主义性质的初级农业生产合作社
C．完全社会主义性质的高级农业生产合作社
D．农村人民公社
E．家庭联产承包责任制

39．毛泽东在《论十大关系》中提出的在经济方面应处理好的关系有(             )
A．重工业和轻工业、农业的关系
B．沿海工业和内地工业的关系
C．经济建设和国防建设的关系
D．国家、生产单位和生产者个人的关系
E．中央和地方的关系

40．中国共产党进行自身建设的特殊困难有(             )
A．中国工人阶级人数较少  B．小资产阶级思想的影响比较严重
C．党的理论准备不足      D．国内外敌人的力量异常强大
E．封建主义的影响比较严重

三、简答题（本大题共4小题，每小题6分，共24分）
41．为什么说毛泽东思想是中国共产党集体智慧的结晶？
42．毛泽东提出的人民军队的性质、宗旨和任务是什么？
43．在人民民主专政制度中民主和专政之间的关系是什么？
44．我国经济建设以自力更生为主、争取外援为辅方针的内容是什么？

四、论述题（本大题共3小题，请任选2小题作答，全部作答只按前2小题评分。每小题13分，共26分）
